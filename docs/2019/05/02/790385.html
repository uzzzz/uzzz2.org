《稀缺：我们是如何陷入贫穷与忙碌的》（美）塞德希尔·穆来纳森

1. 专注红利

试想一下你的任务快到截止日期了

意味着作业要交了/程序再不抓紧要延期了/方案得抓紧写了...

这会带来什么影响？空前的效率，奋笔疾书。在时间充裕的情况下，我们往往三天打鱼两天晒网，但是在明显的deadline情况下，我们的效率会更高。

这就是时间的稀缺性导致的专注红利；所有的拖延症，就是缺乏时间的紧迫感

类似的还有食物的稀缺性的专注红利；对于饥饿者和非饥饿者，食物对于这两者的吸引力是不一样的，饥饿者对于食物，更加专注，大脑被俘虏。

类似的还有金钱稀缺；缺钱者，大脑更多的被金钱占据。

2. 专注之弊

2.1 专注某一件事，意味着我们会忽略其他事
比如：我们在未任务赶工，以便在dealine之前完成，但是我们往往会忽视比如代码的质量，比如久坐的伤害；

2.2 对事物的专注会抑制竞争意识；
当生某人气时，抑制会忽悠优点，专注其缺点，所以会越想越气。

2.3 稀缺会导致人的认知能力下降、削弱人们的分析、判断、逻辑推理能力；

比如在工作时，你在为还没筹集到明天要交的房租发愁，那么你的工作能力会下降、错误率会升高；这是因为你能力不足吗，并不是，是因为你的大脑被其他事情占据；大脑带宽的不足影响了你的能力

2.4 稀缺也会导致人们的执行控制能力下降、削弱他们抑制行为和控制冲动的能力
比如在工作时，你同时在处理焦头烂额的几件事，然而今天的午饭又是糟糕的咸菜馒头，你已经没有能力控制你的情绪了，直接爆了粗口；因为你的大脑被这几件事占据，没有足够的带宽来控制你的言行举止得体。

2.5 带宽负担
正如2.1-2.4所说，我们的大脑的带宽是有限的，一旦被占用，就会挤压其他事情。
就像CPU处理多个程序就会降低转速一样，当有稀缺吸引时，IQ就变差
而且，还会出现注意脱敏的情况，比如节食者看到一闪而过的甜甜圈、鸡腿广告，会瞬间被吸引注意力；或者可以在点外卖或者美食的页面沉醉很久。

所有没有被满足的需求俘获了我们的大脑，成了我们念念不忘的事情

3.权衡

权衡型思维，稀缺资源导致的决策（比如卡路里账本，吃某样必须放弃另一样）

3.1 稀缺不仅提高了失误的成本，也为人们创造更多的机会去犯下错误

比如当金钱紧缺时，我们更容易为了短期的债务，进行高额的借贷。并且这种决策惯性会一直影响我们。
即使我们知道每天进行储蓄，能够在将来摆脱现在的困境，我们依然不会执行。因为我们已经陷入专注红利中，只能专注短期的得失

3.2 一旦意志力不过是去学习如何控制注意力和想法时，就能开始提高自身的意志力了
有意识的行为才能制止自动选择，比如被甜甜圈吸引。

3.3 资源的倾斜

  • 稀缺会导致资源的倾斜，比如对时间的紧张感，会导致付费行为；对金钱的紧张感，会导致以时间来换

  • 稀缺陷阱不限于实物资源的紧缺；问题在于如何花钱和利用时间

  • 稀缺情况下产生短视和借用，会导致为来的利益作为代价，所以坚持储蓄、健身很难坚持的原因 稀缺尤其是搁置，会让你搁置那些重要但不紧急的事

  • 当我们为了解决眼下难题而及其专注时，就无法有效的规划未来；而从当下脱离出来看，你需要具备宽广的视野和一定的认知资源（需要认知能力有一些余闲）

4.绝对和相对

  • 我们在小物品上为了几毛几分都在计较，而对大物品上缺挥金如土；这就是相对

  • 时间也是一样，如果我们拥有了大片的可支配时间，那么对于时间我们就是挥金入土;如果实际不够，我们就会珍惜每分每秒

  • 但是如果很缺钱，不管是大物品、还是小物品，都会计较那几毛几分，这就是绝对

  • 人们更在乎相对量，而不是绝对量，比如洗衣液的瓶口变大，则会倒更多的洗衣液。比如黑暗中的火柴和大白天的火柴

我们富裕的东西就好比瓶盖口，钱多，钱的瓶盖口就大，流出去就感觉不明显。钱少就瓶盖口就小，流出去感觉很大

但是：

  • 专业技能/对某一领域知识的深度理解，能够对认知形成改变，能够更接近绝对，而不是相对，比如音乐家对音程的把握

  • 稀缺也会让我们成为某一领域的专家，比如穷人珍惜没一分钱，富人珍惜每一小时，节食者珍惜每一卡路里，小别胜新婚（珍惜每一次相聚）

  • 穷人更贴近于经济人，有绝对的金钱价格尺度

  • 节俭和稀缺并不一样，节俭是保持一种富裕的责任感，而穷人则会随时保持权衡

5.警示

  • 要提前规划，不要等到最后关头支付昂贵的费用，很有可能还会错过机会（比如等到最后一周再买票，导致买不到票）
  • 对于重要但不紧急的事要坚持做，比如制定计划、阅读、健身
  • 意志力比喻成肌肉，消耗多了会劳累，如果持续的抵制诱惑，那么意志力就会被耗尽
  • 人们以为在资金充足的时候，我们会精打细算，充满远见卓识，事实并非如此，特别在资金充裕的时候，我们容易犯下拖延的毛病。拖延就是对眼下之事盲目乐观;会导致我们就算有点充裕，也会没有足够的带宽来抵制诱惑
  • 孤独者会对一件事情过分专注，孤独者会在他们自认为很重要的事情上表现糟糕;超过阈值，越在乎越容易失去/表现不佳
  • 稀缺造成了一种心态，这种心态让稀缺长存
  • 节省带宽（大脑）带来高额回报（比如各种操作越来越傻瓜式）
  • 余闲的重要性，比如工作安排；余闲可以包容错误，比如有钱人被坑买了昂贵的物品，和穷人被坑，他们能承受的能力不一样。
