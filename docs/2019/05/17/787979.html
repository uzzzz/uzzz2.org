众所周知，Apache Flink 最初是由德国柏林工业大学的几个博士生和研究生从学校开始做起来的项目，2014年，其核心成员开发了 Flink，同时将 Flink 计算的主流方向定位为实时计算并在同年将 Flink 捐赠给 Apache ，后来快速孵化，成为 Apache 的顶级项目。

Apache Flink 是业界公认的性能优异的大数据实时计算引擎之一，除了社区的参与贡献外，还与其商业公司 Ververica（原名：dataArtisans）密切相关。随着 Flink 在不同行业的多样化场景中落地，用户在遇到问题时会寻求社区的答疑与支持；同时，社区也希望好的经验能够更广泛的传播。为此，Apache Flink 的中文社区网站 Ververica（zh.ververica.com）与 Apache Flink 社区微信公众号 Ververica 正式发布。

Apache Flink 中文社区网站简介 

Apache Flink 中文社区网站（zh.ververica.com）是 Ververica (原dataArtisans，Apache Flink 商业母公司）指定的 Apache Flink 中文社区网站，社区网站运营的初衷是希望赋能社区，为 Flink 的爱好者与学习者提供充分的 Flink 技术支持与学习交流的机会，整合社区资源，聚集 Flink 用户，一起打造有活力有深度的 Apache Flink 社区。


Apache Flink 中文社区网站能够提供什么？ 

  • Apache Flink 线下交流学习的机会


Flink China Meetup 由 Apache Flink Community China（以下简称：Flink China） 发起，目前已在杭州、上海、北京、深圳、成都五个城市举办 10 余场，已经有超过 2000 名社区同学参与线下 Meetup 找到 Flink 组织，社区规模在不断扩大，Meetup 版图会延伸到更多城市，更多 Flink Meetup 资讯将会在网站发布。

  • 全面的开发者资料


网站从博客分享、开发实践、专业教程与资料中心四个板块为大家提供 Flink 的相关参考资料。

  • 博客分享：除掌握技术细节外，大家还需要关注 Flink 的未来发展规划与行业应用前景，博客板块涵盖社区 Meetup 与重量级会议精彩的技术分享内容。
  • 开发实践：Flink 应用中如果对某个功能特性存疑，开发实践板块汇聚社区深度开发者对 Flink 核心技术解析，可以为你提供灵感。
  • 专业教程：网站提供基础、进阶、运维、实战的系列 Flink 入门教程，系统地帮你梳理 Flink 的技术 Roadmap ，每周钉钉大群直播，与 Apache Flink 的 Commiter、Contributor 在线互动交流。


除此之外，Flink China 社区 Meetup 的所有嘉宾分享 PPT 以及社区专刊，免费下载，尽在“资料中心”。

  • Apache Flink 一线行业实践案例


Apache Flink 开源至今已经将近五年，社区日益枝繁叶茂，越来越多的企业采用 Flink 作为计算开发引擎解决其实时计算、流计算等问题。我们按照行业、使用场景与应用的深度分类，希望能够为你提供参考与指引。
 




Apache Flink 中文社区网站的建设目标是成为用户参与感强、积极活跃的社区，网站初上线还在不断完善中，欢迎社区同学帮忙测试，提供意见和建议，点击“阅读原文”，一键直达！
