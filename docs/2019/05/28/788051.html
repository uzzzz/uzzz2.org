新加坡公司诉讼主体资格证据公证认证怎么办

所谓新加坡公司主体资格就是新加坡公司的合法开业证明合法注册证明文件，即新加坡公司注册证书，Goodstanding之类的文件。 由于中国没有加入取消文书认证的海牙公约，因此，新加坡产生的文书拿到中国来用必须经过中国驻新加坡使馆认证，方可有效使用。

新加坡公证即由新加坡公证人对新加坡公司文件内容加以确认，新加坡公证以公司书的形式体现，由公证人对文件审核后，将公证书和所附文件捆绑为一体，并在公证书上签章。

新加坡公司在国内投资的越来越多，新加坡公司主体资格要想被承认，就要办理主体资格公证认证。

一家央企和一家新加坡注册的公司要在国内合资投资一个项目，在这期间双方发生矛盾，项目无法继续进行，国内公司以违约为由起诉这家新加坡公司，要求其返回支付的费用。国内法院在受理时要求提供该新加坡公司主体资格公证认证文件。

首先需要对新加坡公司信息进行查册，即到政府登记注册部门调取该公司的信息。在新加坡公司信息都是公开查询的，包括公司目前状态，注册时间，注册地址，董事信息等。

新加坡公司注册资格公证认证的资料包括：

营业执照、章程、股东或董事名册、股东或董事会议决议、董事在质证明、公司存续证明、银行资信证明等等。

新加坡公司注册资格公证认证的流程：

1.新加坡公司文件先委托公证员或者国际公证律师先进行公证；

2.然后再将已经公证过的文件递交到新加坡法律学会进行认证；

3.将已经外交部认证好的文件递交到中国驻新加坡使馆认证。

然后送法律学会认证，最后再拿到中国领事馆认证。
