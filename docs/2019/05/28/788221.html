中美贸易战升级,预测钾肥价格行情后期走势

  • 柚子社区
  • Donate

　　中美贸易摩擦近日升级。自2019年6月1日0时起，对已实施加征关税的600亿美元清单美国商品中的部分，提高加征关税税率，分别实施25%、20%或10%加征关税。钾肥市场弱势持续，成交仍显清淡，市场供应量充足，交投弱稳。钾肥市场供需难以平衡，市场需求支撑不足，价格弱势企稳。硫酸钾市场观望维稳，市场价格挺价企稳，交投维稳，市场暂无明显库压。

　　据海关总署统计数据，2019年一季度中国进口自美国的纯氯化钾、其他氯化钾、硫酸钾累计数量分别为0吨、4.422吨、24.15吨，占上述3类产品1~3月中国进口总量0.0001万吨、295.17万吨、1.30万吨的0%，0.001%，0.19%。从自美国进口钾肥的占比来看，提高加征关税税率的影响不大。

　　中国钾盐钾肥分会秘书长陈国福认为，加征关税对中美钾肥贸易影响有限，原因主要有三点：第一，中美之间钾肥贸易量有限，美国不是中国钾肥主要进口国家。2018年中国从美国进口钾肥469.9吨，而全年中国进口钾肥752.6万吨，从美进口钾肥贸易仅占中国全年进口总量的0.00624%。第二，近年来中国钾肥自给率不断提升。目前，国内钾肥龙头企业供给量保持稳定，这也让国内钾肥市场稳定运行。第三，近期国内春季用肥结束，夏季用肥开始，季节性钾肥需求处于相对淡季。同时，国内港口和边贸库存正常，这也保障了钾肥市场的稳定运行。

　　近期《农资导报》调研了国内钾肥形势，国内市场整体走势偏弱。农业钾肥需求，春耕用肥基本结束，夏季果蔬用肥尚需时日，且夏季作物用肥以高氮肥为主，钾肥需求平平。工业需求，中下游复合肥企业开工不足，原料多按需采购，价格一单一议。

　　国产钾肥，青海盐湖集团生产装置运转正常，盐湖基准产品60%粉晶氯化钾执行价格2350元(吨价，下同)。青海小厂开工率低，成交冷清，57%粉晶1700元左右，地区到站价2050~2100元。港口进口氯化钾库存供应充足，新单跟进不足和经销商报价预期回落致价格小幅下滑。62%白钾港口主流报价2350~2380元，俄红报价2180~2200元，大颗粒报价2340~2400元。边贸市场货源充足，东北市场钾肥需求结束，62%白钾口岸价格保持在2050元左右。
