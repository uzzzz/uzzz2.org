没车牌也能开上国产特斯拉：北京3年租赁每月1万1，全国最低7千

摇不到车牌没关系，特斯拉现在有了新举措。

特斯拉官方最新宣布，即日起为北京客户推出三年免息融资购车与免费车牌租赁服务。

简称融资租赁方案。

特斯拉称，通过融资租赁方案购买Model 3长续航版本与高性能版本，三年免息至少能节省5.3万元，同时三年免费车牌租赁则减免总共约6万元。

具体方案

需要摇车牌的方案，比如北京。

特斯拉称，即使没有北京新能源牌照，也能订购三个版本的Model 3：

  • Model 3长续航后轮驱动版（基础价42.1万元）

  • Model 3双电机全轮驱动版（基础价46.3万元）

  • Model 3高性能全轮驱动版（基础价52.2万元）

但以上都是现金方案价。

如果现在通过三年融资租赁的方式，支付方案如下：

  • 第一，支付12万元保证金；

  • 第二，每月支付1.17万元租金。

按照该方案计算，三年后累计总支付金额为42.12万元（不含保证金），跟买一辆相差不大，只是不用再等车牌了。

另外，面向全国客户还进一步推出最长5年的免息融资租赁方案。

每月最低租金7000元。

车牌、车牌，车牌

当然，如果有车牌，事情自然好办很多。

5月31日特斯拉宣布可预订国产特斯拉后，起售价32万8，与非国产版的Model 3相比便宜4万9。

也推出了融资分期方案，月供最低人民币1100元起。

根据特斯拉提供的金融方案，利率8%按计算，月供想要低至1100元起，需要：

50%首付（16万4），固定1年贷款期限，50%的车价作为预留尾款（16万4）。

如果这样计算，月供的1100元，全是贷款利息。

不过，尾款处理方式有3种：申请展期、一次性还清或二手车置换。

在订购页面默认有租赁和贷款两个选项。

租赁，保证金为9万8千4，36期，利率7.91%，每个月1万出头。

贷款，首付9万8千4，60期，利率8%，月供4600多。

此外，如果你想要有一个中国红（多涂层）车漆、19 英寸运动轮毂、黑白内饰（白色座椅）、完全自动驾驶能力的国产Model 3，你需要掏45万3千5，才能领回家。

国产造车新势力不跟上？

不过特斯拉推出如此方案，对中国车主倒可能影响更长尾。

毕竟竞争激烈的中国市场，造车新势力肯定不会“袖手旁观”。

上周特斯拉宣布预订国产Model 3后，主要几家造车新势力就纷纷表态了。

比如小鹏汽车认为，特斯拉Model3国产版32.8万（不带自动驾驶）毫无竞争力，起码应该再降价1万美元。

小鹏汽车董事长何小鹏还认为，特斯拉在拿了政府这么多政策和补助后还是这个价格，要不然是成本控制有问题，要不然还想获得5年前的利润率。

按照这个价位，“小鹏P7出来肯定辗压特斯拉。”

而蔚来则表态，国产Model 3和ES6存在一定的竞争关系，但蔚来自己还比较有信心。

“ES6在尺寸空间、豪华感等方面有明显优势。在性能、续航上和国产Model 3不相上下。此外，我们相信会提供更好的服务和本地化运营能力。”

最后还有李想的理想智造。

他们没有说太多，只是放出一张海报，一语双关：并“价”齐驱。

即国产特斯拉Model 3的32.8万元定价，跟理想智造首款车型售价一样。

不过上述三家，目前还只有小鹏汽车和蔚来有量产车型，理想智造还需再等等。

而小鹏汽车之前跟神州租车母公司神州优车有投资入股关系，特斯拉融资租赁的玩法，对小鹏可能并不存在壁垒。
