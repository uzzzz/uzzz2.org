被5月GitHub Top20榜单惊呆了! 原来区块链大佬都在做这个...

GitHub 上项目的活跃指数，在一定程度上代表了这个项目的开发状态。


频繁更新代码的项目有可能正处于构建和完善中，而停止更新代码的项目，要么没有进展，要么太完美，不需要更多的更新工作了，但这是一种极为罕见的现象。


在区块链领域，State of the Dapps 从 Github 获取包括来自 Ethereum、 EOS、 Steem 和 xDai 平台开源项目的 commits、pushes、issues 和 pull requests 等数据，在一定程度上反映了区块链行业开发者的真实活跃度。


5月份，Github 上的区块链项目的活跃度排名发生了显著变化，像 ARK、Santiment、AION Network、Raiden 和 Kyber Network 这些在4月份表现很活跃的项目，在5月份都跌出了前20。总的来说，这些项目5月份在 Github 的提交数少于4月。


然而，有个事实不可否定，那就是以太坊仍在区块链行业占据主导地位，在活跃度排名前20的区块链项目中，有17个项目来自以太坊，只有1个项目来自 EOS，其他两个分别来自 Aelf 和 Tendermint。这就表明，开发者仍然钟爱以太坊，以太坊仍为最活跃的区块链社区。



以下是截至2019年6月4日，过去30天内按开发者活动排列的 Top 20 区块链项目，希望老铁们能够通过这些项目，看清区块链开发者们正在做的事情以及努力的方向。


如果你确实想进入这个行业一展拳脚，相信这份榜单是个不错的参考。


1. Status


Star：2671

GitHub 地址：

https://github.com/status-im/status-react


长期以来，号称以太坊手机版 OS 的 Status 项目一直是区块链行业的佼佼者，其最近30天在 Github 上足足产生了7133个事件（event），这就表明 Status 开发团队在开发 iOS、Android 和桌面版 Status 应用的道路上仍在持续努力向前。


2. Cosmos


Star：1131

GitHub 地址：

https://github.com/cosmos/cosmos


Cosmos 是基于 Tendermint 的区块链互操性平台，自主网上线后，Cosmos 以惊人的4245个事件迅速升至 Github 区块链榜单的第二位。随着其原生加密钱包应用 Lunie 的推出，Cosmos 对其 Cosmos Hub 的迭代正在顺利进行。


3. Storj


Star：656

GitHub 地址：

https://github.com/storj/storj


去中心化云存储网络 Storj 在 Vanguard 发布之后运行得很快。该团队还为客户和合作伙伴引入了一个名为Tardigrade的单独版本，它与原来的Storj产品放在一起，后者已经包含了存储、治理和开源协作。Storj 团队的活跃度一直保持稳定，本月大约产生了4084个事件。


4. Aragon


Star：602

GitHub 地址：

https://github.com/aragon/aragon


长期以来，以太坊的明星项目 Aragon 一直运行良好，其平台创建的去中心化组织提出了一个强有力的主张。主网成功运行后，Aragon 团队在过去的30天内向 Github 提交了3292个事件，虽然比上个月略有下降，但仍旧表现出了强劲而持续的增长趋势。


5. Augur


Star：545

GitHub 地址：

https://github.com/AugurProject/augur-core


如果你没有对利物浦赢得欧冠冠军进行加密投注，还说你了解区块链吗？Augur 的去中心化oracle（预言机）和预测市场协议平台是目前运行在以太坊主网上最出色的区块链项目之一。上个月提交事件数目达到了3289个，所有迹象都表明，这个项目仍在蓬勃发展。Augur博客还提供每周更新的开发详细内容。



6. High Fidelity


Star：726

GitHub 地址：

https://github.com/highfidelity/hifi


High Fidelity 是一个虚拟现实平台，由虚拟世界 Second Life 创始人创建，目前已经举办了多场虚拟活动，目前正积极致力于EOS集成，上个月在 Github 上个月提交了2888次事件，比今年早些时候的下跌了25%。


7. 0x


Star：1070

GitHub 地址：

https://github.com/0xProject/0x-monorepo


5月，对 0x 来说确实是一个非常重要的月份，其项目团队在 Github 上提交事件的数量增加了一倍之多，达到了2675次，为4月份的208%。有超过30个项目正利用0x的基础设施在创建各类去中心化交易，总交易次数达到了71.2万次，如今这支区块链行业中最具才华的团队之一是时候做出一些重大举措了。


8. Origin Protocol


Star：430

GitHub 地址：

https://github.com/OriginProtocol/origin


Origin Protocol 正在利用以太坊和 IPFS 创建一个去中心化交易市场，Origin Beta 和手机端 app 已经在以太坊主网上线，你可以用它购买各种稀有的、令人兴奋的、有时甚至是怪异的物品，比如19世纪的小提琴、各类以太坊艺术品以及全世界范围内的奢华旅行租赁。上个月在Github 上有3468次更新，很明显，Origin 团队需要做的事还有很多。


9. Gnosis


Star：628

GitHub 地址：

https://github.com/gnosis/MultiSigWallet


Gnosis 是一个基于以太坊的预测平台，其 Dutchx 智能合约、the Gnosis Safe 和 the Gnosis Olympia Alpha 已经上线，其他项目可以基于此创建属于自己的交易市场，可以说 Gnosis 为区块链社区做了很多工作。在过去的一个月内，其在 Github 上提交的事件高达2602个，这个不断努力的团队将为区块链生态带来前所未有的价值。


10. Aelf


Star：571

GitHub 地址：

https://github.com/AElfProject/AElf


Aelf 作为一个去中心化的云计算网络，是这份区块链Top20榜单内的一个新成员，其在5月份提交了2396次更新。这家总部位于新加坡的团队刚刚基于亚马逊网络服务发布了v0.7企业 Alpha 版本，根据其每周进度报告，其新产品的推出也已提上了日程。



11. Neufund


GitHub 地址：

https://github.com/Neufund


总部位于柏林的 Neufund 是一个通过“股权代币发行”在以太坊区块链上发行 security token 的平台。上个月在其 Github 上共发生了2190次更新，比上个月显著增加，很明显，从头开始构建一个 security token 平台仍需大量工作！


12. MakerDAO


Star：172

GitHub 地址：

https://github.com/makerdao/awesome-makerdao


MakerDAO 致力于成为一个去中心化金融平台，其生态包括稳定币 Dai 、基于 Maker CDP 机制的抵押贷款和社区治理三方面。MakerDAO 不仅仅是以太坊网络中使用最多的平台之一，其团队在5月份也忙得不亦乐乎，仅仅5月份，就在 Github 上提交了 2097 次事件，比上个月增加了近50%。


13. OmiseGO


Star：267

GitHub 地址：

https://github.com/omisego/ewallet


泰国明星项目 OmiseGO 自从上线以太坊网络以来一直在深入研究区块链的可扩展性和互操作性。随着 MVPs 和 Plasma 测试网的投入使用，OmiseGO 团队为进一步推进项目工作在 GitHub 上提交了2093个事件，足足比上个月增加了26%。


14. Trust Wallet


Star：203

GitHub 地址：

https://github.com/TrustWallet/wallet-core


Trust Wallet 是一个安全的多币种钱包，适用于 Android 和 OS 设备，其目前不仅已在多个区块链行业的项目和机构中使用，也是数字货币交易所 Binanc 的合作伙伴。自2月份以来，其增长势头不减，据 GitHub 数据显示，Trust Wallet 在5月份提交了1962次事件。


15. Streamr


GitHub 地址：

https://github.com/streamr-dev


Streamr 是一个数据交易市场的开源平台。目前，Streamr 平台已经处于核心测试阶段，提供包括赫尔辛基（ Helsinki）电车、南特（Nantes）空气质量以及个人谷歌健康等在内的各类数据。上个月在 Github 上的提交量增加42%，达到了1788次，我们期待这个非凡的团队能做出更多的贡献。



16. Blockscout


Star：551

GitHub 地址：

https://github.com/poanetwork/blockscout


POA Network 的 BlockScout 是以太坊生态系统的一个开源区块浏览器工具，允许用户搜索和探索Ethereum和POA网络上的事务、地址和平衡。作为第一次进入这份榜单的新项目，Blockscout 团队5月份在 Github 上提交了1750个事件。


17. Metamask


Star：3216

GitHub 地址：

https://github.com/MetaMask/metamask-extension


几乎每个区块链业内者都知道“Metamask”这个名字，它不仅是一个浏览器上的扩展程序，也是一块以太坊生态系统的基石，是这份榜单的常客。尽管已拥有数百万的下载量，Metamask 团队仍在努力改进他们的产品，5月份在 GitHub 上的提交量达到了1714个。


18. Colony


Star：298

GitHub 地址：

https://github.com/JoinColony/colonyNetwork


Colony 是Top20榜单的新成员，为未来的工作形式提供了一个新平台和新模式，在 Colony 团队的设计中，团队间的协作需要有通证激励。5月底，Colony 发布了路线图中的第一个产品 Glider。为此，Colony 团队在上个月完成了1439次提交，比4月份增加了75%。


19. Golem


Star：2774

GitHub 地址：

https://github.com/golemfactory/golem


Golem 与以太坊有着同样的想法，就是成为“世界计算机”，并且目前在技术和设计方面都处于领先地位。其 Beta 版本已支持下载，通过使用 Golem，用户可以选择贡献或者购买计算机算力来完成大型工作。过去一个月，Golem 团队在 GitHub 上提交了1372次事件，这个项目正在以强劲的增长动力吸引着越来越多的开发者关注。


20. Truffle


Star：7935

GitHub 地址：

https://github.com/trufflesuite/truffle


Truffle，被誉为以太坊的瑞士军刀，是整个区块链生态系统中使用率最高的开发工具套件，是区块链生态系统中不可或缺的增长动力，其开发者活跃度一直保持相对稳定，其团队5月份在 GitHub 上提交了1337次事件，比上一个月增长了2%。


以上，就是5月份活动指数最多的20个区块链项目，其中，20个项目有17个是构建在以太坊网络上，这说明了什么？老铁们，悄悄告诉营长吧！
