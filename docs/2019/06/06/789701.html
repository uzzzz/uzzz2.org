先看效果，这是在MacOS的Terminal上运行的效果

第一步：下载.NET SDK

下载地址：https://download.visualstudio.microsoft.com/download/pr/b4e6bfc6-1843-4154-8b5a-6bd85702df16/270d40a3ede0b72d3f0b801bf103ea96/dotnet-sdk-2.2.107-osx-gs-x64.pkg

下载完后，双加打开

就像安装正常软件那样的安装步骤

第二步：打开一个新的Terminal

在终端里输入

$ dotnet

然后输出就想上面显示的那样，就表示安装成功了

第三步：新建一个简单的C#项目

终端里输入

$ dotnet new console -o myApp
$ cd myApp

dotnet是命令，new console是参数，表示创建一个C#的控制台项目，-o表示创建一个文件夹，文件夹里存储刚创建的项目的代码和所有的文件
然后查看创建项目时，默认创建了哪些文件

$ ls

输出如图

最后我们通过一个命令来运行该控制台项目

$ dotnet run

输出

Hello World!

本教程是.NET SDK Core 2.2
官网地址：https://dotnet.microsoft.com/learn/dotnet/hello-world-tutorial/install
