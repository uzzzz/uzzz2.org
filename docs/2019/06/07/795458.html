每天早到公司20分钟，我会做这三件事情

  • 柚子社区
  • Donate

从实习到现在，不管怎样，我都会早到公司20分钟。虽然比不上那些大佬，五点起床运动，干活，但这20分钟也足够让我做好这三件事情。

01 整理公众号文章

如果留意过我的公号排版的童鞋，会觉得我的排版很简单，没什么特色。第一方面，我比较懒，不想花太多时间去弄；其次，我个人觉得（不喜勿喷），排版只要简单整洁就行，没必要弄太复杂。

基于此，我会使用 markdown 语法来进行写作。我一般会在一、三、五早9点发文章；提前一天晚上我会先用 markdown 把文章编辑好，第二天到公司迁移到公众号中，并进行简单的修改，这个工作比较简单，大约五分钟搞定。

为什么不晚上一次性弄好，因为懒，写完就想上床。

02 阅读公众号文章

我之前就一直会看技术类的文章，但偏向于简书、知乎。

现在开始改看公众号文章，在不同的阶段，人的阅读是有不同偏向的。现在，由于工作和兴趣需求，将类型分为下面三类：

  • office系列，主要以Excel为主，扩充技术，用于工作
  • 阅读，主要是看小说推荐类的，用于兴趣爱好
  • 职场类的，目前探索中

现在不怎么看Python相关的了。这里建议每个类型挑选几个不错的就行，不要关注太多，星标置顶每天必看。这个需要花10分钟时间，看不完中午午休补完。

03 提前进入工作状态

最后，花5分钟，想一下当天需要完成的任务；并打开需要的文件夹和文件，并调到自己合适好操作的位置（电脑小，没办法）；最后开始做起来。

对我而言，提前进入工作状态很重要，上午的效率是最高的，基本上我上午会完成65%的工作量。

04 结语

大家有什么职场心得和小说推荐，都可以留言告诉我哦。
