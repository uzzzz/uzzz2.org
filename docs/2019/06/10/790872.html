白话区块链

从入门到精通，看我就够了！

过去的四十多年里，这个世界经历了一次次迅猛的科技变革。每次变革浪潮过后还能够继续保持领先的科技公司凤毛麟角，而微软则是一家经历了许多次变革后至今屹立在科技界顶端的巨头，不得不令人怀疑这个公司掌握着某种能够大概率准确拥抱并引领变革的秘密。

这一次的区块链变革浪潮中，微软毫不犹豫地成为了区块链与比特币的“头号大厂铁粉”......


 01
微软的过去并非一帆风顺

几乎每一家历史悠久的科技公司面对现代科技的日新月异都如同高空“走钢丝”，一个稍有不慎就可能不可避免地陷入衰退、甚至跌入谷底。

微软当然也不例外，也曾经衰退过，甚至很多人以为它就这样一去不复返......纵观微软的发展史，你会发现这完全就是一个饱经沧桑的巨人。

微软帝国史可分为三段：

第一段：野蛮生长，垄断傲娇

1978 到 1999 年，这是比尔·盖茨的时代

比尔·盖茨把微软打造成为一家高科技垄断帝国。比尔·盖茨带着强烈的求胜心态，想尽一切办法追求市场份额最大化，与当时的科技巨头 IBM 展开了合作与后期的竞争，直至称霸了个人 PC 软件市场。后来，除了 Windows 操作系统，微软又推出了强大的办公软件 Office，以及后来系统捆绑 IE 浏览器获得巨大成功，虽然不久过后便被网景公司指控通过捆绑、预装浏览器是垄断行为。

通过碾压式的垄断，微软每年的营业收入飞速暴涨，从 1993 年开始，比尔·盖茨连续 20 多年上榜福布斯美国富豪榜，做了 10 多年的世界首富。

第二段：收拾烂摊子，陷入绝境

2000年到2013年，这是鲍尔默的时代

这或是微软最坎坷的一个时代，比尔·盖茨的继任者鲍尔默开始了漫长的收拾前一个时代的烂摊子。全球各地针对微软的反垄断纠纷案件不断发酵，搞得鲍尔默焦头烂额，虽然起初仅仅是一些罚款和声誉上的不光彩问题，并没有对巨无霸微软产生太大的影响，微软还是一如既往的高营收。

但是到了后来，我们发现，微软历史上大部分失败的产品都是在这个时代推出的。或许微软继承了上一个时代的高傲，在这个时代里微软推出了最失败的 MP3 播放器 Zune、最失败的 Windows 操作系统版本 Vista、毫无优势可言的 Bing 搜索引擎，但压垮骆驼的是一个原本寄予厚望的移动互联网超级入口 Windows Phone，这也让微软在后来的移动互联网领域里一败涂地。

在这个时代里，微软的股价不断地进入新低，似乎这个科技巨头就会这样慢慢被埋没，被所有人遗忘在历史的长河里。

第三段：吸取教训，尊重对手、开放共赢，重回巅峰

2014年至今，这是新微软时代

2014年，比尔·盖茨卸任董事长，印度裔新 CEO 纳德拉上任，在这个时代里微软自上至下、甚至连企业文化也发生了重大的变化。有人说，是纳德拉拯救了这个快速衰退中的巨无霸，并把它带回了上升轨道，重回市值第一。

微软与区块链、比特币的“情缘”，也就是在这个时代中开启的......


 02
率先布局区块链，5 年磨一剑

在国内、甚至国际上都鲜有人提及“区块链”这个概念的 2014 年，微软就对这个新事物充满兴趣，并成为第一家真正布局区块链的国际科技巨头。

在过去的 5 年时间里，微软提交的区块链成绩单如果列出来，一定会令所有区块链领域里的竞争对手汗颜：

2014 年 12 月，微软首次为在线商店添加了比特币支付选项，走出了切入虚拟金融业的第一步。

2015 年 11 月，微软推出并启动了 Azure 区块链即服务（BaaS），覆盖全球 24 个地区，推出数款开发工具并吸引了大量新成员，同时 Azure BaaS 平台升级为当时全球最大的认证区块链市场。

2016 年 6 月，微软宣布启动 Bletchley 计划，这是一个被 Azure 所驱动的“开放的、模块化的区块链组件，用微软自己的架构方式创建区块链企业生态联盟”，解决了早期的跨行业区块链使用者的大量常见问题。

2017 年，微软动作更加频繁：1 月，微软发布区块链合作伙伴战略；2 月，与全球各行业代表性企业一同加入企业以太坊联盟，并对以太坊区块链技术进行了大量的投资；5 月，发布区块链概念验证框架、增添了对更多区块链协议的支持与 R3、Chain 和 BlockApps 等建立了伙伴关系；8月，推出企业级开源区块链基础平台 Coco 框架、加入 IC3 （密码货币和合约倡议）计划；10月，推出“Azure 政府机密”（Azure Government Secret）服务。

2018 年 2 月，微软宣布微软身份验证（Microsoft Authenticator）应用程序将整合基于区块链的去中心化 ID 验证技术：DID。

......

对于区块链的支持，微软时任金融服务科技战略主管的 Marley Gray 说道：“我们差不多赌上了整个云服务”。在微软的第三个时代里，纳德拉的重大改革部署中，移动和云服务正是微软当时的两个重点，而 Azure 云服务则是微软区块链战略的核心之一。

可以说是这一次，微软毫无保留地拥抱了区块链。

反观国内一线科技巨头，最早涉足区块链领域探索的是腾讯在 2015 年成立了区块链团队，但从近年来的产出结果看，发展区块链的投入和进展并不是很快。另一个巨头阿里巴巴，则在 2016 年才开始布局区块链领域。

其实想要对比微软对区块链的支持以及腾讯、阿里对区块链的支持，很简单，看他们的云服务主业菜单业务把“区块链”服务放置在什么位置就知道了：

微软 Azure 把区块链类目放置于产品菜单下显眼位置，并且提供的服务琳琅满目，区块链市场内的服务也是数量众多。

阿里云则把区块链划分到一个极不起眼的位置：产品——企业应用——区块链，腾讯云的区块链菜单放置也是类似的归类，可以想象区块链服务在两大国内一线巨头整个云服务布局里的重要程度并不高，并且服务提供上也显得迟缓、落后。

或许现在腾讯、阿里巴巴正在努力追赶，并也已经有了一定的成绩，希望未来他们能够把区块链行业的影响范围拓展到全世界。

 

 03
大刀阔斧的改革，“大船”掉起头来毫不犹豫

其实不少人很好奇，为何一个如此成功的巨头，拥抱区块链这种“要革中心化商业命”的技术，还拥抱得那么剧烈？

这一点就又得回到微软帝国发展史的第三段了，吸取了前两段旧微软时代的深刻教训，新上任的 CEO 纳德拉对微软进行了大刀阔斧的改革，不但虚心与竞争对手展开合作（甚至帮老对手苹果特别定制移动 Office 套件），提出新时代的核心战略：移动和云服务。

云服务需要众多开发者加入共建，总结移动互联网时代最重要的两点就是开放和开源，于是微软停止了固步自封，从过去的“软件强制野蛮捆绑和垄断”到现在的开放共赢，拥抱开源。

2018 年，微软 75 亿美金巨资收购了 Github 这个全世界最大的开源代码托管、社交平台（区块链开源项目基本上都把代码托管到此平台）。一开始，众多开发者用户强烈抗拒这一起收购，认为这是一个“顽固、封闭的垄断老怪物”收购了他们的乐园，但后来他们发现，新微软拿出了最大的诚意，表示将保持 Github 的独立性，不会对它进行干预。

其实，新微软的目的是放下原本旧微软高高在上的架子，把姿态放到最低，完全拥抱开源试图以此吸引更多的开源软件开发者使用 Azure 的云服务，入驻平台参与平台开放共建。随后，Github 平台上的开发者们，开始慢慢放下成见，相信 Github 始终能够保持中立，并开始接受它的母公司新微软满怀诚意抛出的橄榄枝。

目前，就微软拥抱开源这件事，全科技行业都竖起了大拇指，佩服微软这种拥抱开源、开放共赢的巨大转变给整个大行业带来的巨大影响。

 

 04
对于比特币，比尔盖茨向左，微软向右

对于比特币，比尔·盖茨曾经发表过不止一次的看法。早前，比尔·盖茨曾经称赞过“比特币是令人兴奋的”，但最近的一次是 2018 年的 5 月份，他说：“比特币是最笨的投资。对于比特币，如果有最简单的方法，我赞同做空它”。

然而这一年的早先时候，也就是 2018 年 2 月，微软却宣布了一件令全球科技界人士都感到兴奋的事情：微软的一个去中心化身份项目 DID 将成为该公司区块链战略切入点。很快在同年年底，微软官网公布了《去中心化身份》白皮书。或许非技术圈外的人看着没什么，但整个科技界都对 DID 感兴趣，因为它很可能对互联网的未来产生重大的影响。

2019 年 5 月份，微软突然宣布了名为 ION 项目的网络早期预览版，今年晚些时候将在比特币主网上正式运行（该项目是一种类似闪电网络的比特币二层网络来实现的），众多科技界人士纷纷表示，此举可能影响整个科技行业。

什么是去中心化身份（DID）？我们在微软的去中心化身份发布页面找到以下描述：

Microsoft 认为每个人都有权拥有其数字身份——它以安全、私密的方式存储所有个人数据。此 ID 必须无缝集成到日常生活中，让你完全掌控数据访问和使用情况。

按微软的说法，在考察了多种去中心化身份系统后，最终把目光锁定了“公共区块链”，他们认为区块链技术和协议非常适合用于实现 DID。尽管微软公司财大气粗、人才济济，但他们任然选择将这个对公司极具战略性意义的项目放在比特币主网上运行，不但说明微软对比特币网络的安全性和可靠性的信任，更表明了微软对比特币网络价值的直接肯定。

就在 ION 项目的消息发布后不久，有人发现微软的 EXCEL 软件中悄悄将比特币符号列入了货币分类选项。Windows 系统所自带的 ARIAL 字体的（全球使用最频繁的一款英文字体）最新版本中也添加了比特币符号，让大部分的设备都能够正确显示出这个符号。

 

 05
后记

比尔·盖茨时代里，微软科技帝国奠定了坚实的基础，但也埋下了诸多隐患。随后种种大起大落，让微软充分认识到了滚滚潮流逆不可挡，科技、互联网的发展与进化方向似乎冥冥之中已有定数，从互联网最开始的信息的自由传递到现在的点对点去中心网络中价值的自由传输，这完全是一股由人类骨子里透出的“神秘力量”在推动着，就算强如微软帝国也无法抗拒，一切都向着人类自由文明的方向有序推进着。

此时此刻，也许与几十年来科技行业发展进程息息相关的微软深知，就算自己不干，也必定会有别人抢着来顶替他们的位置去干。

在未来，我们可以预见的事情是，微软对区块链的超早期深度布局必将给新微软带来巨大的增长，让它在区块链变革浪潮中始终保持引领者的地位，可见区块链、比特币对微软非常重要。点此链接，收听比特猫日报FM：微软应用区块链技术实现去中心化身份

而微软这个科技巨头的入局，对区块链和比特币来说，也一样重要。没有这个巨头的引领和全力投入带来全科技行业的争相效仿跟进，或许区块链也不会如此快的被推进到全球科技界的视野里，有了主流巨头的加持，“区块链无用、比特币无价值”的论点，不攻自破。

现在的微软早已不是代表比尔·盖茨意志的那个微软，它是新时代的微软。或许在不久的将来，微软通过对区块链浪潮的把握，又会展现出全新的形态。

主流已来，我们拭目以待。
