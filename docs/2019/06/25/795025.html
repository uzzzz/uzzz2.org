AnyQ 搭建
参考https://github.com/baidu/AnyQ文档
使用docker搭建百度开源框架AnyQ
1、拉取docker镜像

# paddle官方镜像
docker pull paddlepaddle/paddle:latest-dev
# paddle国内镜像
docker pull docker.paddlepaddlehub.com/paddle:latest-dev

我采用的是官方镜像，国内镜像没成功

2、运行镜像将镜像映射到端口上

  docker run -dit -p 0.0.0.0:9999:8999 paddlepaddle/paddle:latest-dev

3、进入到镜像中

docker exec -it 75742cdb2f72 /bin/bash

4、在docker中将项目clone下来

git clone https://github.com/baidu/AnyQ.git

5、进入AnyQ

cd AnyQ

6、新建build文件

mkdir build

7、编译

cmake ..
make -j 4

8、下载jdk

wget http://anyq.bj.bcebos.com/jdk-8u171-linux-x64.tar.gz
#解压
tar xzvf jdk-8u171-linux-x64.tar.gz
#添加路径
export PATH=`pwd`/jdk1.8.0_171/bin:$PATH

9、构建索引配置

# 获取anyq定制solr，anyq示例配置
cp ../tools/anyq_deps.sh .
sh anyq_deps.sh

# 启动solr, 依赖python-json, jdk>=1.8
cp ../tools/solr -rp solr_script
sh solr_script/anyq_solr.sh solr_script/sample_docs

10、运行run_server

 ./run_server

11、可以通过最开始映射的端口来请求服务
注意用127.0.0.1可能会出问题
