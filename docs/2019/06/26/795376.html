作为程序员，我经常在各大论坛和博客看到类似：哪个程序好一点？学c#还是学Java，写asp.net有前途吗？php跟java哪个找工作更容易？Python和php区别是什么？

作为过来人，我认为编程的思想基本都是相通的，不同的只是语法不一样，无论java转c#，还是从c#转java，其实都没有多大的困难，但是建议大家如果真心想在程序员这个领域深耕，那么努力学好一门语言，无论php、java还是c#，都有很多公司使用，所以无论你学好哪一门，只要精通了，都可以找到一份很不错的工作。

TIOBE 近日公布了2019年5月编程语言排行榜，TIOBE 编程社区指数是编程语言流行度的指标，该榜单每月更新一次，指数基于全球技术工程师、课程和第三方供应商的数量。包括流行的搜索引擎，如谷歌、必应、雅虎、维基百科、亚马逊、YouTube 和百度都用于指数计算，下面为来自TIOBE的排行榜单：

总体排名变化不大，排名前十的依旧是：Java, C++, C++, Python, Visual Basic .NET, C#, JavaScript, SQL, PHP 和汇编语言。

历史排名（1988-2019）

通过这个趋势表，Java、C、c++基本稳定在前三的位置，Python已经成为最大赢家。

尽管它不代表语言本身的好坏，但从一定程度上能够帮助开发者们了解到编程语言的趋势。

大家再选择开发语言时候一定要有限考虑使用场景，如web开发，建议选择asp.net（c#）或php，如果做游戏开发的，c#的使用率就高过java，所以不一定排名高的就一定适合你。
