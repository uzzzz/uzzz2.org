曾经，缺钱的时候，我傻傻的在百度搜过“怎么赚钱”。

我不知道有多少人跟我这样傻傻的干过，但是我确实这样干过，那是在2015年夏天的晚上。

但是现在回想起来，我一点儿也不傻。相反的，现在的我庆幸当年那么傻过。

毕业三年后，我依然是个朝九晚九的苦逼程序员，每天为了实现策划的各种需求而掉头发，每天为了学习新的技术而思考到深夜。曾经为了修复一个莫名其妙的bug整整一周没有出过公司的门，媳妇都跑到公司，怀疑我快人间蒸发。

就算如此苦逼的工作，一个月下来刨除生活费，也就攒个小几千块钱。

16年2月14日，我休假。晚上十点钟，双手揣兜，一个人在街上漫无目的的闲逛。

看着灯红酒绿的大城市，看着身边经过的奔驰宝马，看着一个个光鲜亮丽的路人，我沉默了

这么繁华的都市，我何时才能有辆自己的车、房？何时才能自由的支配自己的时间，何时才能闲下来，去看看外面的世界？

我想要一份闲适、不费力、每天工作几小时、有自己充足的时间玩儿乐的工作，这是我的目标

我并不想每天坐班，甚至加班到很久很久

那晚的我失眠了，翻来覆去睡不着，打开百度，鬼使神差的敲下了“怎么赚钱“四个字

“色流“，这是我接触的第一个词，也是这个词，在往后的三年时间里，直至今天，完全的占据了我的生活，乃至生命。男人都好色，女人都爱美，这是千古不变的主旋律。

这让我回忆起了高中时代，一帮玩伴儿整晚的通宵，前半夜打游戏，后半夜集体开始看小电影

要玩“色流”，就要有自己的网站

程序出生的我，很快就锁定了自己的目标

“cheng ren网站“，肯定是犯法的，弄不好会把自己现有的生活白白搭进去，那么介于se qing与合法之间，我确定了自己的网站类目：《美女图文》这本身是不犯法的，只是做一个网络的搬运工。比如某某校花、微博上某些极品的妹子、qq群里遇到的放浪形骸的女孩子、某某美女写真集等，这些都是我收集的目标。当然，这些也都是男人们感兴趣的话题

确定好目标之后，就着手操作，说起来也简单，就五步而已，其实蛮艰辛的

  1. 挑选一个域名
  2. 购买一台云服务器
  3. 搭建网站程序
  4. 写内容
  5. 挂广告收钱

首先选域名，很多短域名跟好的域名已经被域名商人们注册完了，剩下的就是些长域名跟不太好记的域名，费了好大劲，才选了跟自己稍微满意的域名，然后付款，6.9刀一年.选择域名商也是很有讲究的，国内要备案，备案不过也麻烦。

选服务器这就费劲了，腾讯、阿里、京东 等国内大厂商随便选都ok，ping值 在50ms-90ms 之间。国外就不同了，因为我不想备案，所以为了选一个ping 值在200ms内的服务商，我是把国外的知名商挨个测试了一遍。国外的服务商基本都支持30day无条件退费。半个月里，我的visa信用卡尽是消费、退款的出帐入账记录，最终确定了服务商，一次性600刀买了半年。

那接下来是网站程序，直接选了wordpress + lnmp 环境

上面这三步费了我俩周时间，主要是测试云主机费时间。至此，我的个人网站、就算搭建完成了，接下来就是网络搬运工的工作了。

我们人类喜欢原创的、更新稳定的、新奇的网站。同样的百度蜘蛛也喜欢这些。 所以我要让百度尽快收录我的网站的话，就必须每天原创+稳定更新 若干文章。

白天上班，偶尔抽时间寻找搬运的内容，记下网址，晚上回家了，在结合初次搜到的讯息，多个网站的信息加以补充，修改成自己的文章，发表。有时候白天没时间的话，所有的工作都回堆积到晚上进行。这样真的很累，上班累费脑筋一天，下班还要在各种网站里搜，提取自己想要的信息，继续费脑子。现在我回想起来那段时间，脑子里天天充斥这各种诱惑的美女，还要稳下心神完成手头的工作，精神与肉体的双重折磨下，天知道我是怎么撑过来的。

半个月后，百度开始收录我的网站了，一个月后，网站日IP达到有1000+了

三个月后，日IP3000+，开始挂广告联盟，每天收益50+RMB。如果照这样下去，最起码，我的网站收入与支出会持平的，一个月1500的广告费，我的前期投入700RMB都回来了

意外来的比我想象中要快。三半月的一天，我习惯性的打开自己的网站，发现网站跳了下其他网站，然后再跳回了我的主页，心里咯噔一下，尼玛，被黑了？？？去看了下百度收录，我去，一堆的我网站里压根没有的文章，还全是英文，各种国外的卖产品的广告。同时，我的网站打开速度也变慢了。确定，一夜之间，我被黑了！！！

辛苦了尼玛三个多月，白干了，而且才回本了200快？！瞬间萎靡了。直接给老板请加俩天，理由：心情极度不美丽。

各位看官，看我写了这么多，写到我破产，会不会不想看了？

其实我也不想干了，老老实实的上班，老老实实的做个普通人，多好。

但是我不服啊，我不要天天挤公交、我不要朝九晚九，我不想自己的时间不由自己支配！！！俩天请假，再加一天周末，爽爽的放松了三天，睡觉！！！昏天暗地、日月无光那种睡，实在是太累太累。

周一，上班，老老实实的那种。周二，开始思考为何被黑。周三，着手学习各种防黑的手段。

一学就是一周。一周内，把云服务器系统重新做了一遍，然后防黑手段挨个验证挨个启用。

一周后，继续开始搬运工的角色。

因为我的域名已经有些基础了，这次收录一周就开始有新的收录了，日ip恢复也快，半月后就恢复到日IP3000+，收入也稳定在50+RMB了。

其实我写的也有些累了，做一下总结吧

半年后，我的网站日ip 9000+，广告收入稳定在300+，招了个兼职文案，我基本只负责网站的安全维护与文章内容审核，一个月光广告联盟的收入就在1w+，其他自己上门打广告的都不算的。前后，折腾了半年时间，我基本已经实现我的时间自由了。因为第五个月的时候，我把本职工作辞了，哈哈。

这里就会有人问了，一个月不是1w+吗。标题是骗人的？一个网站1w+，那三个呢？

对于网上其他的“兼职日入300+“的那些骇人听闻的网赚，兼职？300+？呵呵

没有相应的底蕴堆积，没有99%的汗水洗礼，何来高枕无忧的生活与RMB
