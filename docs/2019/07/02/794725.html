WAY2BIT宣布BORA Island数字内容与钱包平台进入试运营

  • 柚子社区
  • Donate
  • 提供UNIT5的Push Sushi，Find Bird，Snake Worm等游戏内容
  • 利用钱包功能可对‘BORA Token’与‘SHELL Token’进行综合管理及交易
  • 根据反馈改进服务，以提高稳定性和可用性

6月2日，区块链数字内容平台BORA（CEO Cindy Lee）宣布BORA Island服务将进行试运营。

本次发布的BORA Island将提供任何人都可以轻松使用的区块链数字内容与用于数字资产管理及交易的钱包功能。在BORA Island试运行期间，还将提供官方合作伙伴游戏开发公司UNIT5的Push Sushi、Find Bird、Snake Worm等3个游戏作为初期内容。只要是BORA会员，都可以体验这些游戏。后续在经过稳定化运营之后，还将计划与合作伙伴一起推出不同领域的各种区块链内容。

此外，BORA Island还提供了钱包功能，可以综合管理及交易基于以太坊的‘BORA Token’与BORA平台内数字内容加密货币‘SHELL Token’。 BORA Island钱包提供了一个直观的用户体验，BORA Island的钱包在交易Token时，无需掌握复杂的钱包地址，而只需掌握邮件地址，即可方便传输 Token，而所有的交易内容可通过BORA的区块链数据信息探索器BORA EXPLORER来进行确认。

BORA Island目前正着力通过用户反馈来改善及稳定运营服务，故用户暂时只能使用Test BORA Token、 Test SHELL Token。本次发布的BORA Island可通过PC（https://island.boraecosystem.com/ ）或下载手机App（暂时只支持安卓版本）注册会员使用。与此同时，为庆祝BORA Island的试运营，用户能够参与多个活动，提供服务反馈，并获得BORA代币作为奖励。

BORA的CEO Cindy Lee提到：‘我们非常高兴能够发布作为BORA核心服务之一的BORA Island。’，‘我们将着力开展服务稳定化工作，并且计划与各种合作伙伴一起策划、开发并最终推出在日常生活中能轻松使用的各种区块链内容服务。因此希望得到大家的鼎力支持。’

另一方面，除了BORA Island之外，还向内容开发者与用户提供有作为测试网的BORA LAGOON，针对开发者的区块链开发工具及管理空间BORA ATOLL以及BORA区块链数据信息探索器BORA EXPLORE服务。

更详细的内容请参考BORA Ecosystem（https://www.boraecosystem.com）官方网站或者BORA Medium（https://medium.com/boraecosystem）。

■ BORA 介绍

区块链数字内容平台项‘BORA’是以双重区块链结构与独有的Token机制为基础，旨在与各类企业合作，分销数字内容和娱乐服务并提高用户参与度的Dapp平台。目前BORA已经与韩国主要的游戏开发公司合作，如4:33、ENP GAMES、NEON Games、Unit five、Super tree、emong games、pier.6，及与行业合作伙伴Greenwood Networks（房产）、wisepeer（音乐）、Huray Positive（健康）、RS EDUCONSULTING（教育）、Infoseed（节点定位）等展开了合作。此外，BORA还得到了KAKAO games的投资，并以此为契机与KAKAO的区块链项目Klaytn建立了官方合作关系，与其团队密切合作以改进他们的区块链技术，并将BORA平台与Klaytn主网连接起来。
