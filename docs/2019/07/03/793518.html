清华毕业程序员国企干了14年，去应聘小公司，看到工资后愣了

  • 柚子社区
  • Donate

现如今，很多人会用“温水煮青蛙”这样的词来形容那些在国企上班的人。本来很多人在进入国企之前，都是名校毕业的天之骄子，但是因为国企的工作环境，早九晚五，没有压力的时间太长了，相比那些刚开始就在那些互联网公司天天996的员工，竞争力只会越来越差。

最近在职场论坛上看到这样一个帖子：清华毕业程序员在国企干了14年，辞职后去应聘小公司，看到薪资后愣了，还以为看错了！这是怎么回事呢？

image

原来这帖子是一家公司的HR发表的，她说昨天有个清华毕业的程序员来自己公司面试。在简历上，该HR看到这个程序员清华毕业，以前在国企上班了14年，今年已经38岁了。

可是，谁知道这个员工除了那些最基础的代码知识，稍微难一点的问题完全不会。而那个程序员其实要的工资也不高，仅仅只是一万五而已。当时这个HR就感叹：一个有着14年工作经验的清华学生，居然只要了一万多的工资，关键技术不行还被淘汰了，说起来让人遗憾，也让人心酸，难道国企的员工都是这样的吗？

image

有网友说该老员工以前肯定是在混日子，而且在工作过程中也没有学习，所以才会出现这样的现状，找不到工作，工资低也只能怪自己。职场上不是说你是清华的就一定会给高工资，也要看你能力。

image

还有网友说38岁其实也不晚啊，要知道清华毕业的人，学习能力肯定不会太差。而且他要的工资也不高，完全可以给个机会，相信这样的老员工其实会更珍惜现在的工作，创造的价值也不会低。

image

还有网友说真的是废了，14年前的清华毕业生其实含金量是很大的。所以说不要以为进了国企就高枕无忧，不需要学习了。不管在哪里，只有提高自己能力才是最稳定的，也是最不需要担心工作的。

image
