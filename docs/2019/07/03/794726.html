大家好，

炎热的夏天到了，与此同时，大家期待已久的BORA ISLAND服务已于7月1号正式上线。为了服务的稳定性，BORA ISLAND计划先进行一段时间的试运营，之后再进行功能多样化及稳定性维护等更新。今天我们就来简单介绍一下BORA ISLAND以及它未来的发展方向。

将区块链融入日常生活—BORA ISLAND

BORA ISLAND是一个提供了多种区块链内容的平台，用户可以在上边享受他们的日常生活。对于传统的数字内容而言，使用内容服务的用户其数字资产所有权无法得到有效保障。用户为此而付出的时间与努力也就难以得到与之相匹配的合理价值。

BORA的区块链技术提出了解决这一问题的策略。例如，如果你在一款游戏中积累了游戏点数，而这款游戏后来停止了，那么这些点数就可以在其他游戏中得到维护和使用。

在BORA ISLAND中，你会发现BORA提供了各种各样的BApps(区块链应用程序)。在此之前，开发者可以应用区块链技术和测试他们的数字内容在BORA Lagoon上，并且BORA Atoll提供了运行区块链服务所需的开发工具包。

BORA LAGOON测试网指南（https://medium.com/boraecosystem/bora-lagoon-guide-35fad08b6a08）

在BORA，用户即使没有掌握区块链相关的专业知识，也能在一个安全的、基于区块链的环境中与数字内容交互。以易用性和可采用率为重点，BORA将提供各种新的数字内容服务。

注册、创建钱包一次搞定！


为使用区块链APP并有效管理数字资产，用户就需要属于自己的个人钱包。而在BORA ISLAND，当用户注册时，同时就会生成一个钱包，这个钱包提供了一个直观的界面，以查看和管理用户在BORA ISLAND的数字资产。在简单的会员注册后，用户无需繁琐的钱包创建流程，就可以使用BApp。

BORA钱包支持多种资产交易服务，例如收发BORA Shell，这是一个专用于BORA平台的加密货币系统，用于享受广泛的BORA Island数字内容。用户还可以将从BORA内容中获得的BORA Shell转化为可以从平台外部提现的BORA代币。

安全又简单的交易

执行交易很容易。与传统的区块链钱包不同，用户不需要知道复杂的公钥地址就可以发送资产。因为交易只需要邮件地址，用户就能向别人发送数字资产。所有的交易内容都可以在BORA Explorer上查看，它记录了BORA区块链的所有区块信息。我们将在下一篇文章中更详细地讨论如何使用钱包。

BORA Explorer：（https://explorer.boraecosystem.com/?network=testnet）

重要提示：

为了通过用户的反馈改善及稳定服务运营，我们暂时只可以在BORA ISLAND平台和钱包中使用测试用BORA币及BORA Shell。所以在试运营期间，请勿把主网的BORA代币发送至当前的BORA ISLAND 钱包当中。

具有各种内容的BApp

在BORA ISLAND中，您可以用一个账号体验各种领域的BApp。另外，在BORA平台所提供的内容均以BORA Shell作为基础货币来使用。因此，用户可以用在特定BApp所获得的Shell来体验其他的BApp等，进而更加有效地使用自己所持有的数字资产。

了解应用区块链技术的数字资产：NFT（https://medium.com/boraecosystem/boras-nft-non-fungible-token-story-part-3-f646faec4d8e）

目前在BORA ISLAND可以体验BORA的合作伙伴UNIT5的Push Sushi、Find Bird、Snake Worm等游戏，我们计划后续将陆续增加包括游戏在内的网漫、音乐、教育、健康、地图等各领域的区块链服务内容。

BORA平台专用币BORA Shell

目前在BORA使用基于以太坊的BORA代币（ERC-20）与在BORA平台内使用的代币BORA Shell这两种货币。BORA代币是可以通过外部交易所转化为现金的币，BORA Shell只在BORA平台中可用，并且通过访问BORA内容来使用和获取。用户可以使用BORA ISLAND钱包在BORA Shell和BORA代币之间进行转换。不同于直接使用基于以太坊的BORA代币，BORA Shell的交易无需手续费，因此可最大限度地降低了用户负担。

了解BORA的双重币结构：( https://medium.com/boraecosystem/introducing-boras-first-patent-technology-cf6c1d703909)

未来，各种内容服务和零售店将被添加到BORA生态系统中，从而支持使用BORA Shell进行交易。在接下来的BORA ISLAND指南Part.2中，我们将提供有关如何使用您的钱包的更多信息，并最好地利用您对BORA ISLAND的访问权限。
