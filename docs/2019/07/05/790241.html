苦心人，天不负——PMP备考心得体会

为期三个月的PMP备考终于结束，5A的成绩总算是给这段辛苦的岁月画上了圆满的句号。最初了解到PMP考试是在刚入职场时，那时候懵懵懂懂，有同事考过了PMP发了朋友圈，看到全英文的证书上面大大的名字感觉好洋气。

之后一直忙于工作，一直到今年6月份，才终于下定决心要报考。一方面是因为我工作在外企，工作中经常需要支持项目管理团队的一些工作，所以自己也心生好奇，很想系统地对项目管理的内容进行一些学习。另一方面，也是想给自己充充电，为以后的职业生涯规划添砖加瓦。

 

从今年6月份开始进行了三个月的备考，一开始就心心念念要考5A的我也终于如愿以偿。 有一些心得想跟大家分享：

 

选择好的良心机构真的很重要。不得不说机构的PMP培训课程非常值，老师也非常尽职尽责。在报名以后，班主任老师会发出清晰的日程表，包含各阶段需要完成的学习任务，三次模拟考试的时间以及目标，发的学习资料可以怎么使用，等等。

可以说只要报名以后跟着老师的安排一点一点完成，过是肯定没有问题的。班主任老师在整个过程里也会持续跟我们有电话或者微信沟通。一模二模考试我的成绩都很不错，远远高出目标线，但是在三模考试时陷入了瓶颈，焦虑的很，多亏了班主任老师给我打电话给我积极的反馈，让我放平心态积极备考。

2. 一定要有科学的规划。在前期看书时候我总是一脸懵逼，砖头厚的书完全看不明白是在说什么。但是我始终相信读书就是要先把厚书读薄，再把薄书读厚，所以一直配着视频教程看书，一直到第三遍才读出了这本书的精髓，颇有武侠小说里的茅塞顿开之感。关于做题，我也只做了机构给出的题目。三模的题每一套都做了三遍，去细细揣摩题里面的思路和考察的重点，培养题感。事实证明题不在多，思考和总结往往比题海战术更有效果。

3. 应考时候一定要沉着冷静。考试前一天我就去看了考场，了解了电梯和厕所的位置，还坐在里面待了一小下感受考场的氛围。考之前老师说可以准备一些吃的，中间上厕所时候可以来吃点东西补充一下体力，结果考试的题特别难，出来上厕所我都是一路小跑，根本没时间吃。

考试题目的难主要在于题干很长，并且答案里面有很多模棱两可的答案，我心里凉的想肯定得重考了。在考场上赶紧给自己做心理建设，不管怎么样也要挺过去，所以沉下心来冷静作答。现在相信，当时确实有点慌了，但是还好后期冷静下来，不然心态上如果先输了就肯定过不了了。

 

4. 独学而无友。非常感谢机构为我们组建的学友圈，在里面认识了不少认真学习备考的大神，遇到问题也总是积极帮助我们解答。心情低落的时候互相鼓励，真的是一起走过了一段难忘的时光。

回顾这三个月的报考，我推掉了所有的应酬，在加班的间隙和生活的夹角里争取时间备考。成绩出来的一瞬间还是觉得，值了。上天总是眷顾踏踏实实积极进取的人们，而我们滴下的每一滴汗水，终有一天会以别的方式给我们以报答。
