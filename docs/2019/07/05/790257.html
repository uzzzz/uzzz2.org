1. 下载Scala

首先下载Scala，可以去官网http://www.scala-lang.org/ 下载当前最新版本(2.13),

也可以直接下载2.12版本http://www.scala-lang.org/download/2.12.2.html

注意安装Scala前，请安装配置好JDK， 详见https://blog.csdn.net/u010590120/article/details/94736800

下载好的文件在 目录 、home/你的机器名/下载 目录下。

为了统一起见，键入命令mkdir /opt/scala 在opt下建立目录scala，注意后续操作 大多需要在root权限下操作，所以键入命令su root 输入密码获取root权限

cp scala-2.12.2.tgz /opt /sacla将其复制到opt/scala目录下

2.解压Scala

键入命令cd  /opt/scala 来到文件加下 ，键入命令 ls -al 可以看到复制过来的文件

键入命令tar -xvf scala-2.12.2.tgz 解压文件

3.配置环境变量

键入命令 vim /etc/profile 打开配置文件

按键i 进入编辑模式

在文件末尾输入

#scala environment
export SCALA_HOME=/opt/scala/scala-2.12.2
export PATH=$PATH:$SCALA_HOME/bin

配置结果如下所示

按键 esc 进入命令模式，键入：wq! 保存输出

3.测试

键入命令 source /etc/profile 使配置文件生效

然后 键入 scala -version 测试是否配置成功，若成功显示以下信息则表示配置成功

 

键入scala 进入scala 解释器 便可以进行交互式编程
