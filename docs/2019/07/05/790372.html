Guardian是世界领先的在线金融商品交易平台，在华经营已超过10年，是专业外汇指数交易商。Guardian以优质的客户服务，技术支持以及同业最出色的交易系统使其成为客户理想的选择对象。Guardian服务对象包括金融机构客户，对冲基金，经理人账户和个人客户，多年来以卓越的信誉赢得了客户的满意。 

Guardian集团是专注于为全球客户提供包括外汇、贵金属、期权、指数、数字货币等交易产品，以及基于移动终端提供交易服务的综合类服务商，通过完善的服务及产品设计为客户提供更加周全的服务。

2009年成立至今，已经拥有超过 500,000名客户， Guardian集团已经发展为国际知名的投资公司，业务范围涵盖欧洲、澳洲及亚洲等主要外汇市场。Guardian现有150多名专业人员，都是来自全球外汇经纪人才库的精英和佼佼者。除了专业知识，我们还提供全球30多种语言客户支持。

 

Guardian的平台优势

Guardian集团是一家专门从事银行间清算业务的外汇及CFD差价合约交易的经纪商，提供银行间流动性、STP直通式处理模式以及系统搭建等技术支持服务，为帮助客户能够快速进入全球最大的交易市场，提供最佳的市场流动性及交易环境，Guardian集团先后成立S-FIC清算中心、组建Quants团队、建立 GuardianFOREX交易中心、GuardianOPTION交易中心。我们有最全的产品，最好的流通性，为了交易更加安心，我们提供权威有力的资金监管，更多的交易选择，为了提供产品多样的交易服务，让交易更加便捷，我们具有稳定快速的技术支持。“为客户打造公平、透明的交易市场环境，提供可靠的交易平台。

作为综合类金融服务平台，Guardian拥有以下四大特色。

    第一、强大的S-FIC清算中心：S-FIC清算中心有报价稳定的IBX数据中心，顶级银行及投行报价源；透明的订单执行，与客户无利益冲突；市场对冲渠道，深度市场流动性 FIX API / MT4 Bridge；多边清算通道，无重复报价及STP直通式交易模式。清算中心专注为机构客户提供多边清算通道，同时接入一级银行报价，并提供精密的计算体系以及完整的交易技术解决方案，服务包括外汇经纪商、投资机构、对冲基金以及其他市场参与者等全球行业相关机构，持续为其提供稳定的一级银行报价、极低的点差、完全透明的市场环境及订单执行。

      第二、权威的监管机构：选择一家外汇平台最重要的因素是确保资金的安全性，而具有正规的监管机构是保证资金安全的重要前提，监管机构需要对外汇平台商的资金情况、运营情况、财务状况、偿债能力等因素进行极其严格的审核，只有达到标准后才有资格获取牌照。Guardian与世界顶级银行合作，资金独立存放，全方位保证账户安全；完善出入金流程－我们只接受来自账户户主本人的出入金操作请求，不支持来自他人的的操作请求，最大可能的保证您的资金安全；账户层级审核制度－根据入金量及出金量的金额设定相应级别，由不同级别人员进行审核，确保资金流向安全；合理杠杆选择－外汇交易风险巨大，为此，我们针对交易经验量身推荐合理杠杆，最大限度的帮助合理交易；精准后台评估系统－为了保证能够第一时间掌握市场情况，我们提供专业、完备的市场行情信息。

      第三、强力的资金安全保护机制：三大资金保护机制。一、负余额保护，余额自监控系统及风控系统确保您的账户余额永远不会低于初始存款水平，保证您的亏损永远不会大于账户余额。二、安全便捷出入金，为保证交易的安全性，我们只接受账户户主本人的出入金操作要求，不接受来自第三方的出入金要求，并且，您的独立资金银行会对每一笔出金请求做严格的财务审核，以确保出入金的安全。三、独立存放资金，所有零售客户资金与公司自身资金完全分离，并存放于世界顶级银行账户中，客户为交易所投入的保证金只作为正常交易使用，不得用于其他任何用途。

第四、采用金融级尖端科技：作为综合类金融服务商，Guardian为您提供稳定的交易系统及创新的科技。优化您的交易体验，我们从未停步。Guardian通过FIX API连接了10多家主要银行以向交易者提供最优的市场流动性、订单执行力及价格反馈，支持一键下单平仓，同时支持EA操作及追踪止损功能，及时锁定利润Guardian连接EQUINIX美国机房为客户提供稳定的运行系统，能够提供更快的处理速度和更短的加载时间，充分保证客户接收来自顶级流动性供应商提供的充足流动性，最大程度上为您提供稳定、快速的交易系统。Guardian有强大后台系统及软件支持、稳定便捷的执行系统、功能齐全的交易系统、极速交易系统和多级代理及IB返佣系统。
