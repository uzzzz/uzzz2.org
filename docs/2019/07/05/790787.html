搭建ubuntu + python + Flask的服务器框架

 

安装ubuntu 18.04 LTS

我们在本地用vm安装一个ubuntu 18.04 LTS来作为服务器，这个安装不涉及到开发，所以就省下1000字了。

安装python

Ubuntu 自带了python 3.6.8，所以又不需要安装了

安装Flask

sudo apt install python3-venv

venv 模块是一个虚拟环境，这样，您可以在一台计算机上拥有多个不同的 Flask 环境，并在每个项目的基础上安装特定版本的模块，而不必担心它会影响您的其他Flask安装。

接着我们开始创建

mkdir my_vvvip_app
cd my_vvvip_app
python3 -m venv venv
source venv/bin/activate

现在虚拟环境已激活，您可以使用 Python 包管理器 pip 来安装 Flask：

pip install Flask

在虚拟环境中，您可以使用命令 pip 而不是 pip3 和 python 而不是 python3。
使用以下命令验证安装，该命令将打印 Flask 版本：

Flask --version
(venv) hlx@ubuntu:~/my_vvvip_app$ flask --version
Python 3.6.8
Flask 1.1.0
Werkzeug 0.15.4
(venv) hlx@ubuntu:~/my_vvvip_app$

环境OK了，那我们来测试一下。在项目的目录下创建一个hello.py文件

from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Life is perfect only when it is green!'
export FLASK_APP=hello
flask run

上面的命令将启动开发内置服务器。

输出将如下所示：

在 Web 浏览器中打开 http://127.0.0.1:5000，您将看到 “人生有点绿才完美！” 消息。

要停止开发服务器，请在终端中键入CTRL-C。

完成工作后，通过键入 deactivate 来停用环境，您将返回正常的 shell。
