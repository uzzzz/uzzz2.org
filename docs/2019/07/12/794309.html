准备工作

一般这样的都可以用这种套路安装：

 

过程演示

把源码移到Linux机器上，解压：

tar -zxvf jdk-7u80-linux-x64.tar.gz

root用户修改/etc/profile

添加如下配置：

export JAVA_HOME=/XXXXX/XXXXXX/XXXXXX/jdk1.7.0_80
export CLASSPATH=.:%JAVA_HOME%/lib/dt.jar:%JAVA_HOME%/lib/tools.jar
export PATH=$PATH:$JAVA_HOME/bin

重新加载下这个文件：

source /etc/profile

这个使用root用户java和javac都可以用了，并且如果有bash的也可以了。

 

cshrc配置工作

修改本用户下的 .cshrc，添加如下配置：

setenv JAVA_HOME /XXXX/XXXX/XXXX/XXXX/jdk1.7.0_80
set path=($JAVA_HOME/bin $path)

重新加载下这个文件：

source .cshrc

此时Java和Javac都可以使用了！
