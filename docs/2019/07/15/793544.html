此安装程序必须使用 Java 开发工具包 (JDK) 执行, 但 C:\Program Files\Java\jre1.8.0_172 不是有效的 JDK Java 主目录。

前言

安装jar时遇到这个问题，以管理员身份运行
注：着急了直接看解决，跳过检查

检查 JDK1.8

打开此电脑>属性>高级系统设置>

JAVA_HOME没问题（检查这是不是你的jdk安装路径）
往下拉，打开Path（不是上边的Path）

加入%JAVA_HOME%\bin上移到最顶部（有了上移，没了添加上移，这个路径利用了我们的JAVA_HOME）

配置成全局变量，让Java文件那都可以执行

解决

再次执行的你的命令
如果执行还是老样子
则先进入你电脑自带jdk \bin目录下 然后java -jar 执行你的jar包就可以了
