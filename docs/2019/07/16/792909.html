博主上半年因为课程少（其实就是闲）的缘故，向各大中小 IT 公司，如字节跳动，360等的实习生岗位投递了简历，广撒网，多敛鱼，择优而从之。最终，博主看中字节跳动的高待遇——一天400，就近租房补贴，免费的一日三餐与下午茶打动（这谁顶得住啊），毅然决然地投入字节跳动的怀抱！谁拦不住我，我说的！如今，我已经离职，特意撰写此博客总结我的实习生活，并希望对大家有所帮助！

面试

首先，当然是介绍大家最紧张的面试环节了。由于自身的研究方向是自然语言处理，因此博主投递的基本上都是各公司的 AI，机器学习，nlp 等相关的实习生岗位。此外，博主阴差阳错的进入了字节跳动的后端部门，不过也是 AI 相关的，而且钱多啊，也就无所谓了。

字节跳动

字节跳动近几年飞速发展，甚至直接对标充钱巨头——腾讯。无论是否成功，字节跳动还是很大气的。它在北京有许多工作地点，如中航广场，学清嘉创大厦等，而楼主的面试与实习地点就在学清嘉创大厦。

进入学清嘉创大厦，过了门禁，同学们就可以直接向前台小姐姐表明实习来意了。前台小姐姐在你登记完个人信息后，会给你发送笔试题，并引导你前往面试间答题。字节跳动一般会在大厦的一楼大厅专门设置有面试间以方便面试，一共有一轮笔试，两轮技术面试以及最终的 HR 面试（可电话面试）。废话不多说，敲黑板！！！接下来，就是满满的干货了。

字节跳动的笔试形式是三选二，答题时间为45分钟（很充裕），没有面试官，语言不限——编程语言，伪代码，自然语言都可。而博主拿到的面试题，

  1. 多边形周长均分——有 n n n 个点 f 1 ( x 1 , y 1 ) , f 2 ( x 2 , y 2 ) , … , f n ( x n , y n ) f_1(x_1, y_1), f_2(x_2, y_2), \dots, f_n(x_n, y_n) f1​(x1​,y1​),f2​(x2​,y2​),…,fn​(xn​,yn​)，它们构成了 n n n 边形，并且相邻点的连线是水平线或者垂直线。现在，我们需要将该 n n n 边形 均分为 k k k 份，并给出等分点的坐标；
  2. 链表求和——给定两个非空链表来代表两个非负整数。数字最高位位于链表开始位置。它们的每个节点只存储单个数字。将这两数相加会返回一个新的链表；
  3. 接雨水——给定 n n n 个非负整数表示每个宽度为 1 1 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

说到这，想必不少同学都感到十分熟悉了。没错，上面的题目大多都是 Leetcode 上的原题或者稍加改动的版本。大家可以自行在 Leetcode 上查看原题，并自己动手解题。当然，要是没有思路，可以看看博主的博客文章（打个小广告）。

  1. 多边形周长均分，没找到，所以，嗯，还是参考我的博客吧;
  2. 链表求和，Leetcode 第445题;
  3. 接雨水，Leetcode 第42题;

其实，根据博主的经验，无论是像字节跳动这样的大公司，还是处于创业阶段的小公司，很多 IT 公司的笔试题大多可以在 Leetcode 上找到。因此，刷题还是很重要的！

接下来就是两轮面试了。前面说到，由于阴差阳错，博主误投了字节跳动的后端部门，而博主的研究方向是自然语言处理，所以导致面试官不知从何入手。尴尬！不过，面试官十分亲切和蔼，并没有因此为难我。在这里说一句，字节跳动的风气很好，同学们（字节跳动互称同学，不用敬语）都平等相处，交往友善，十分 nice，疯狂打电话！

好了，说回原题。由于专业不对口，面试官专业知识没有问多少，我也没装逼，在那不懂装懂。注意，不懂的就是不懂，别在那不懂装懂，因为这只会弄巧成拙，使自己的印象分大打折扣。如果你在某处遇到问题了，可以直接和面试官交流，不用担心紧张。面试官会十分耐心的引导你一步步解决，如果实在没有思路，就直说，面试官会换成其他问题，也不会为难你的。总之，不要装逼！

后来，面试官还是问了我一些编程题，如二叉树的左右子树互换等等，都挺简单的。所以说，代码能力很重要！代码能力很重要！代码能力很重要！至于想了解题目解题思路的，老规矩，点击链接 二叉树的子树交换 。

所幸，博主的代码能力还过得去，编程相关的题目都顺利解决了。所以，虽然专业不对口，面试官还是给我通过，并大为称赞道，小伙子，未来是你的！

至于最后的 HR 面试就更不用担心了，一般电话面试即可，大多是谈谈薪资，实习时间等等。所以，一般技术面试通过的话，获得 offer 的概率就很大了。不过，听说会有 HR 把你当备胎，并且一直拖着你，十分恶心，比如某 m m m 公司。而字节跳动的 HR 会主动和你联系，而如果有冲突，也可以调整时间，并且态度很好，小姐姐声音好听，绝不拖拉。这里，我还是要为字节跳动的办事效率和办事风格疯狂打电话。

对了，在你面试之后，无论是否通过，都会收到字节跳动的短信，询问你对字节跳动面试的感受。而且在进入字节跳动后，我发现公司会专门制作宣传片希望员工注重面试时的态度等方面。可见，字节跳动对人才的重视程度！

实习

总之，博主还是很幸运地进入了字节跳动，开始了自己在字节跳动的实习生涯。

工作篇

具体的工作内容就不介绍了，还是说说工作制度，工作环境之类的吧。刚进入公司时，每位新人或者实习生，会有一位mentor指导你，帮助你快速融入公司。但是，公司没有专门的培训部门，只能自己看文档，和mentor沟通。

字节跳动的工作制度是大小周——第一周工作六天，第二周工作五天，如此循环反复，弹性的996——早上10点30，晚上8点30，而且经常会有员工加班至深夜（大多是自主加班）。当然，员工只要能完成工作，早点走也是完全没问题的。字节跳动的同学的代码能力还是很强的。说回996，怎么说呢？就我个人而言，还是能撑住的，因为在学校就经常泡在图书馆，但是我还是坚决反对996的——都没时间找小姐姐了。不过由于我是实习生，所以无需全勤，但是工资和补贴是按照出勤天数发放的。

说起小姐姐，字节跳动的工位安排还是很科学的。字节跳动会将经常有合作的部门的工位安排在相邻位置，而不是一大帮程序员扎堆。所以，小姐姐还是很多的，虽然都和我没关系罢了。

此外，字节跳动的工位既有存在隔板的那种，也有无隔板的桌子，比如博主的。前者有更多的私人空间，后者更方便交流。而且每个工位都会摆放一个盆栽和空气净化剂（你懂的）.


说回工作，在字节跳动，每名员工都要定制双月目标（实习生好像不用），必须是具体的，如产品上线，功能开发完毕等等。此外，基本上每个团队每周都会有周会，实习生也是可以参加的，介绍每周的工作进度。

对了，在字节跳动，正如我前面所说的，员工们互称同学，或者直接称呼名字，不使用敬语，如哥，姐，总之类的。这是一鸣同倡导的字节跳动员工的精神气质，字节范—— 追求极致，务实敢为，开放谦逊，坦诚清晰，始终创业。所以，你可以在公司的内部沟通软件，Lark上直接联系一鸣同学。当然了，我没试过。顺便说一句，字节跳动内部的沟通软件和内部文档做的十分棒，让人爱不释手。

至于工作设备，字节跳动会提供 Thinkpad, MacBook Air, Macbook pro 这三种机型供你选择。是你，你选择哪一款？而且字节跳动会提供显示屏——戴尔的1080p，可升降，无边框。此外，前端的同学还可以申请苹果的显示屏，真的是爽啊！

待遇与福利

再来说说大家最感兴趣的话题——待遇与福利问题吧。在字节跳动，实习生（研发）的工资是按照出勤天数计算的，一天四百。而且与正式员工相同，字节跳动会为实习生提供就近租房补贴，一个月1500，但是这是上限，也会按照你的出勤天数进行计算的。此外，租房必须距离你的工作地点步行时间不超过30分钟，以地图软件为准。总的来说，在实习生的薪资方面，字节跳动几乎碾压国内全体 IT 公司。

在公司附近租好房了，怎么解决一日三餐呢？别担心，字节跳动帮你解决。公司会为每位员工提供免费的一日三餐，而且种类丰盛，味道鲜美。早餐从9点开始，有包子，混沌，面条，面包，糕点，香肠，小菜，豆浆，粥，牛奶以及各种小吃。到了正餐，则除了主食，还会有虾，牛羊肉等令人食指大动的美食。听说，每位来字节跳动工作的同学都，胖了。

更“令人发指”的是，字节跳动会有专门的后勤同学们精心为员工准备下午茶，如蛋挞，炸鸡翅等等，并且用小推车来到你的身边。这是博主每天工作时最期待的环节，因为可以光明正大的摸鱼。至于零食什么的就更不用提了，都是小意思。


怎么办？这样真的会胖死的。没关系，字节跳动还为每名同学准备了健身房。公司一般在大厦内部设置健身房，或者在临近的商场为同学们提供免费的健身服务。所以，燃烧你的卡路里吧！

每逢节假日，字节跳动还会为大家提供精美的礼物。

总结

最后做个总结吧，在我看来，字节跳动还是很 nice 的，优点多多，也有不足，但是瑕不掩瑜。

优点

  1. 薪资高，福利好；
  2. 工作氛围好；
  3. 的确能长见识，锻炼能力；

缺点

  1. 996！996！996！
  2. 没有专门的培训部门；
