我们是BORA团队。今天我们将介绍BORA ISLAND的第一个数字化内容并展示这些区块链Apps（BApps）是如何运用BORA SHELL的。我们将解释如何在该平台上使用SHELL，并讨论这些新BApps的一些独特和吸引人的特性。让我们开始吧！

在“Find Bird”中帮助寻找从农场逃出的可爱小鸟们！

Find Bird是一个在游戏规定时间内在众多种类的鸟中找到正确的鸟的游戏。随着游戏难度的提升，鸟的数量与种类也在增加，在规定时间内找到正确的鸟变得更具挑战性。您将在可爱复古的画风与鸟充满个性的舞蹈中体验别样的趣味。

在玩游戏过程中，当您通过一关或发现一只新鸟时，您将会得到cookies作为奖励。

您可以使用cookie在商城中购买道具。这些道具可以帮助您取得新的个人最佳记录。如果您想购买道具用于创造新的游戏纪录，却没有足够的cookie，该怎么办呢？

有时候，如果您很幸运的话，可通过抓捕偶尔出现的小偷来获得额外的cookie或者SHELL。

您也可以使用BORA SHELL来购买所需要的cookie。当然，玩家可以从自己BORA钱包或把从在BORA网络中的其他BAPPs中所获得的SHELL发送至Find Bird来购买cookie。

Push Sushi

Push Sushi是利用尽可能少的移动次数来推开寿司的智力游戏。游戏中存在数以千计的难题与不同的难度，玩家可以选择适合自己的难度来进行游戏。

在Push Sushi游戏中，玩家可以获得SHELL作为游戏奖励。SHELL还可用于购买可以帮助玩家通关的道具。

Snake Worm

Snake Worm是一个您要躲避大眼睛病毒同时还要使您的蛇虫生长的游戏。蛇虫不断地向前移动，您可以通过触摸屏幕来改变方向。这个游戏看着很简单，但它很难去躲避不断出现的大眼睛病毒。

在游戏中收集SHELL的同时也要让您的蛇虫继续生长，您将获得新的种类的蛇虫作为奖励。让我们看看谁先填了获奖清单。

此外，您还可以参加一个锦标赛，向其他玩家挑战额外的SHELL作为奖励。继续挑战您自己吧！

透明、安全的交易记录

在BApps中使用SHELL的所有交易记录，都可以在BORA ISLAND的交易内容中进行跟踪。点击Block Information按钮，即可前往BORA EXPLORER。在下面的例子中，这位玩家检查了EXPLORER，发现他们在玩了一个Push Sushi游戏后收到了10个SHELL。

您可以在Push Sushi 游戏服务的钱包地址（From field）中查看发送（Amount）至玩家钱包地址（To field）的10个 SHELL的信息。以及相关交易的日期与时间等详细内容。通过透明地提供所有交易信息，BORA创建了一个更安全、更值得信任的内容生态系统。

用玩Push Sushi获得的SHELL在Find Bird购买cookie！

这个简单的例子展示了BORA平台强大的功能与灵活性。像Find Bird与Push Sushi一样，我们计划BORA ISLAND的BApps都以BORA SHELL作为内容中的使用货币。每个应用程序都是唯一的，所以他们可能都有不同的获取和使用SHELL的方法。但是由于货币是通用的，用户在整个平台上使用它的方式就具有更大的灵活性。这也BORA的一个服务——在区块链上记录及保障数字资产所有权。

BORA团队将继续与合作伙伴合作，提供更加丰富的区块链服务内容。随着SHELL的使用范围逐渐扩大，我们鼓励社群能够在BORA ISLAND中尽情地使用SHELL。
