NEO将与Ontology强强联手，打造开放性跨链平台

                                          When we connect, magic happens

今日，全球两家领先公有链项目 NEO 与 Ontology 宣布达成合作，双方将共建一个开放性跨链平台，打造下一代互联网基础设施。随着区块链大规模应用时代的到来，此次合作将结合双方优势，共同推出一项高效包容的互操作性协议，加速整个行业发展步伐。鉴于双方在区块链行业中的领先地位，此次合作将成为整个行业的一个里程碑。

在本次合作中，NEO 与 Ontology 将基于自身技术特性进行全面适配，以充分应用互操作性协议。NEO 将聚焦支持各类数字资产的协议与组件开发，Ontology 将继续发力于面向未来的去中心化身份框架。

这一突破性的互操作性协议拥有以下重要特征：

·  生态友好：该协议专为跨链同步设计，既不会发行代币，也不会有一个专门的智能合约体系，从而保证成员链之间良性竞争与协作；

·  准入门槛低：该协议旨在促进技术无缝接入，现有的区块链项目无需开发或调整协议层即可便捷使用；

·  交易原子性：该协议旨在实现跨链交易的终局性与原子性，并重点关注跨链智能合约交互，以扩展去中心化应用的应用场景；

·  安全性增强：该协议将在技术和运行层面加入一系列复杂的机制，以增强跨链交易与交互的安全性。

谈及此次合作，NEO 创始人，Onchain 创始人、CEO 达鸿飞先生表示：“此次合作不仅预示着 NEO 与 Ontology 的未来方向，也代表了整个区块链行业的未来。双方通过此次开创性的合作，共同解决行业难题。通过打造全球化的跨链平台基础设施，双方将共同应对挑战，为现实世界的问题提供区块链解决方案。随着下一发展阶段的到来，我们希望通过此次国内两家领先公链项目的精诚合作，共建下一代互联网基础设施，加速整个行业发展与进步。”

本体创始人李俊先生认为：“开放的全球化的跨链体系与平台，可以在更大范围拓展跨链的价值交换，业务流程协作等，以构建下一代 Internet 的基础设施。Ontology 和 NEO 在公链领域的持续的技术积累和社区拓展，会让这个体系在不同行业中获得更大范围的应用，真正让区块链服务更多的实体行业。”

此次 NEO 与 Ontology 达成的深度合作开创了业内先河，双方将引领区块链发展进入一个新时代。据悉，互操作性解决方案白皮书将在2019年第三季度发布，开放性跨链平台将于同年第四季度上线。随着平台的发展，双方将携手共建下一代互联网，最终真正实现智能经济。
