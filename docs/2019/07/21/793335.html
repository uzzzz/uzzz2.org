单击右键选择打开终端，输入以下命令进行更新

sudo apt-get update
sudo apt-get upgrade

也可以将软件源换为国内源，加快下载速度，我这里换为清华源
cd /etc/apt/

将原来的cp sources.list文件备份为 s.list 以便日后换回来
sudo cp sources.list s.list

打开sources.list文件 ，替换为如下内容
sudo gedit sources.list

deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial main restricted universe multiverse 
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-updates main restricted universe multiverse 
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-backports main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ xenial-security main restricted universe multiverse


保存，关闭 ，再次更新一下
sudo apt-get update
sudo apt-get upgrade

安装git
sudo apt-get install git
安装vim
sudo apt-get install vim-gtk
安装pip
sudo apt-get install python-pip
升级pip
pip install --upgrade pip
发现版本19.1.1可用 ，对/usr/bin/pip 进行修改授权
sudo chmod 777 /usr/bin/pip
打开文件，修改如上图 所示
gedit /usr/bin/pip
修改完保存并关闭，执行以下命令升级
pip install --upgrade pip==19.1.1

同样地安装pip3
sudo apt install python3-pip
sudo chmod 777 /usr/bin/pip3
gedit /usr/bin/pip3
pip3 install --upgrade pip==19.1.1

安装python-opencv
sudo apt-get install python-opencv
验证
python
import cv2
最后安装cmake,我选择cmake-3.8.2-linux-x86_64.tar.gz版本 ，建议用迅雷下载
下载后解压到自己的目录
打开~/.bashrc
gedit ~/.bashrc
文件末尾添加：export PATH=/home/cjx/cmake-3.8.2/bin:$PATH
/home/cjx/cmake-3.8.2/bin是我自己的目录
保存，source使它生效
source ~/.bashrc
查看验证
cmake --version
