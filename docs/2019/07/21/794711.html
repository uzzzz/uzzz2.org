Docker 08 - 本地镜像发布到阿里云

本地镜像发布到阿里云的流程 :

镜像的生成方法 :

  1. DockerFile
  2. 从容器创建一个新的镜像
docker commit [OPTIONS] 容器id [REPOSITORY[:TAG]]
# OPTIONS说明：
## -a :提交的镜像作者；
## -m :提交时的说明文字；

将本地镜像推送到阿里云 :

  • 本地镜像素材原型

  • 阿里云开发者平台

网址 : https://www.aliyun.com

  • 新建仓库镜像

  • 将镜像推送到 registry
# 官方推荐拉取和推送的方法

# 1. 登录阿里云Docker Registry
$ sudo docker login --username=l_fyyyyyyyy registry.cn-qingdao.aliyuncs.com
用于登录的用户名为阿里云账号全名，密码为开通服务时设置的密码。

# 2. 从Registry中拉取镜像
$ sudo docker pull registry.cn-qingdao.aliyuncs.com/llffyy/centos:[镜像版本号]

# 2. 将镜像推送到Registry
##请根据实际镜像信息替换示例中的[ImageId]和[镜像版本号]参数。
$ sudo docker login --username=l_fyyyyyyyy registry.cn-qingdao.aliyuncs.com
$ sudo docker tag [ImageId] registry.cn-qingdao.aliyuncs.com/llffyy/centos:[镜像版本号]
$ sudo docker push registry.cn-qingdao.aliyuncs.com/llffyy/centos:[镜像版本号]
