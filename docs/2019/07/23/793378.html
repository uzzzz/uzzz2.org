重装系统（一）：准备启动盘及镜像文件

1.注意事项:

  • 启动盘制作时会对u盘进行格式化，请将数据备份好之后再开始操作

2.相关提示:

  • 普通格式的U盘一次性传输数据在4GB以内，NTFS格式的U盘可以传4GB以上的。系统镜像一般
    在4GB以上，所以推荐使用NTFS格式
  • 系统镜像推荐下载64位的
  • 浏览器上镜像下载速度较，推荐使用迅雷进行下载
