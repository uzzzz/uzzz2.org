1）Robot Global钱包是什么？

答： Robot Global本质上是一个基于区块链+AI技术的大数据云服务平台，目前最核心的应用是Robot去中心化数字钱包，用于用户管理、存储数字资产，是Robot最基础的应用Dapp。数字钱包是进入币圈的一个基础应用。用户可以进行币币交易、转账支付等功能，同时支持多币种存储，采用冷热钱包分享技术为用户提供一个安全存储的环境，在未来将会有更多的应用开放。

2） Robot Global 的技术团队是谁？

答：Robot Global是由原亚马逊和甲骨文公司的技术团队综合开发，融合了亚马逊的人工智能深度学习与甲骨文的数据库管理系统，打造了一个超强的前后端系统程序工程。通过高频量化交易及高效的市值管理，向全球用户提供市值管理、安全储存+智能收益的去中心化服务平台。

3） Robot Global 钱包与主流钱包有什么不同？

答：Robot Global在各项特性与技术层面来说，与市面主流钱包有着相同的去中心化牲，在原有的存储功能上，Robot Global开创了Robot理财服务，让用户不公可以享受到安全的存储空间，同时还可以获得AI智脑带来的稳健收益，还将有更多的理财DAPP应用上线。

4）Robot Global是不是骗局？会不会跑路？

答：Robot Global是基于区块链技术打造，第一阶段会在以太坊上生成，第二阶段将打造属于自己的公链，区块数据可通过以太坊浏览器查询，用户存储的资产都在区块上，除了私钥持有都任何人不可操作或者交易。
