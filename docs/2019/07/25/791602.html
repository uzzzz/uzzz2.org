Redis|Redis 分布式实践——Redis Sentinel 哨兵

作者：郭朝

Redis Sentinel（哨兵）是用于监控 Redis 集群中 master 和 slave 状态的工具，是官方推荐的高可用性 (HA) 解决方案，Sentinel 从 Redis 2.8 版本开始才正式生产可用。Sentinel 系统可以监视一个或者多个 Redis master 节点，以及这些 master 节点的所有 slave 节点，当某个 master 节点下线时，自动将该 master 下的某个 slave 节点升级为 master 节点，替代已下线的 master 节点继续处理请求。
