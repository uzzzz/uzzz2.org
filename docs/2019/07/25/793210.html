1. info 

docker info : 显示 Docker 系统信息，包括镜像和容器数。。

语法

docker info [OPTIONS]

实例

查看docker系统信息。

$ docker info
Containers: 12
Images: 41
Storage Driver: aufs

 Backing Filesystem: extfs

CPUs: 1

2. version 

docker version :显示 Docker 版本信息。

语法

docker version [OPTIONS]

OPTIONS说明：

  • -f :指定返回值的模板文件。

实例

显示 Docker 版本信息。

$ docker version
Client:
