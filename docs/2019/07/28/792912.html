领导暴怒发言：技术就是古代的叫花子，贱命一条！

近日看一网友分享的群聊天截图，貌似是个领导在群里“暴吼”，给大家梳理一些内容关键信息：

 

来自网友分享的截图

为什么人家可以做到半年不发工资依然如此为xxx卖命，而你才几个月没发工资？

 

你们这点小命值几个钱？说难听点就是古代的叫花子，贱命一条。

 

才通宵几个就一个个叫困，至于吗？

 

哪怕现在就把整个技术部开除了，分分钟招来一帮愿意全天28个小时在公司工作的优秀员工，且个个比你们出色，还不发牢骚。

 

看你们一个个贱婢样的，身在福中不知福。

 

每天加班到晚上10点我们会给你点外卖，别的公司哪家能做到？

 

加班到凌晨3点可以第二天下午来上班，试问还有哪家公司如此出类拔萃超群绝伦？

 

 

不知大家看完什么感觉？我当时竟无语凝噎。。。

不发工资还有理了？是忘了马云爸爸说过的名言了吗：

员工的离职原因林林总总，只有两点最真实：钱，没给到位；心，委屈了。

 

而且从一个创业者角度来说，发不起工资就是领导者的耻辱。这点羞耻心都没有了，干脆别开公司也别做领导了。

还没到什么国民级成功人士级别就浮躁得不行了，看谁都是“贱婢”？

真的麻烦一些朋友找工作的时候可长点心吧，去了这种公司的程序员，算是倒了八辈子霉。

这里也请那些“跪久了站不起来”的程序员们醒醒！你的人生不只有赚钱，在这一类公司或者领导底下工作，你也永远不会成为一个大人物的，格局就摆在那儿。

马云说：心中无敌者，无敌于天下。

总之，看不惯就“干”，别懒！懒你就完了，有些资本家怎么看基层员工的还不明白吗？清醒一点！

只有自己努力了，有真本事，才能随时都有底气说：我去nm的，lz不干了！

化悲愤为动力，是每个人都应该学会的能力。
