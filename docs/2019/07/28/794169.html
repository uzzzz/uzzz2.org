  我所在的小区生活配套比较丰富，楼下的餐饮很热闹，程度不亚于一个大学附近的美食街，所以在上班下班的一路上也能够看到有些饭店新开张了，有些饭店在重新装修，商场的起起伏伏，都在生活中。

   而对每个人来说，生活习惯和饮食爱好都不大相同，所以热闹的一条街能够满足各个阶层的需求，而今天想和大家聊聊的是水果店，不知道大家注意到没有，一个小区附近的水果店还是比较密集的，少则1-2家，多则5-10家都有可能。 

   楼下的一个水果店引起了我的好奇，所以今天打算用非专业的角度来做下分析。 

   先说下分析的动机，楼下的这两家水果店是隔得太近了，之间就隔了两个商铺，相隔不超过5米，最开始的那一家在这里很久了，夏天卖水果为主，在冬天卖干果为主，还搭配一些日常的蔬菜，生意还算不错，不过对他们的一些意见是在水果的质量方面，夏天水果储存的周期短了，很多水果会坏掉，而在这方面他们没有重视，可以看到很多水果都不新鲜了。也是接连几次买的水果不够新鲜，慢慢疏远了这家店。 

  而没过多久，在这家店不远处就开了一家新的水果店，这家店的经营方式比较新颖。从开业第一天起就使用了会员制度，即充值xxx元送xxx元，或者干脆送你一些实物。而且水果的质量是很不错的，第一天买的草莓，那质量真是好，所以算是带着一丝欣喜的购物体验，有了这样的体验之后，后来去的次数也多了起来。他们主要经营的是水果，也附带一些零食，水果属于中高端的档次，所以单次购物的成本要比原来高不少，而他们尝试的新模式很不错，即在购物之后，邀请你加入一个微信群，刚开始还觉得奇怪，一看人数都好几百了，到了平时下午或者周末的时候，总能够看到群里有人在询问水果的情况，原来他们还附带送货上门业务，这样一来也算是打开了局面。 

   这是在早期的一些情况，总体来看，第一家店的生意是大受影响，因为经营方式比较传统而且不注重水果质量，导致丢失了一些潜在的客源。 而第二家有了一些创新的尝试，在早期来看效果还不错。    

   随着时间的推移，我充值的几百块很快就用完了，而后续充值的金额也高了一些，但是显然这种情况下的购物成本是变大的，比如原来买一次水果大概是20-30左右，而现在大概是在50-70元左右，所以这种情况下充值的额度是很快会用完的，这种情况下我也会刻意的减少购物的次数。 

   人都是追求实惠的，对我来说，不一定追求是进口，比如新鲜大个的车厘子，一个单价40，一个单价80，从实惠的角度来看，我会优先选择40的，所以从长期来看，大家的想法似乎都是差不多的，第一家的生意似乎也慢慢开始好了起来，我已经几个月没有去过了，今天去看了下，发现他们也开始做了改变，原本的水果质量不佳，现在进去，两家的水果布局都是很相似的，而且水果的价位上，第一家是从附近批发的比较新鲜的水果，看来有了之前的一些教训，他们有了经验也开始做了调整。 

   所以在这种情况下，我有些担心第二家的经营状况了。 所以我也在思考几个问题，第一个是改进模式，第一家开始是落后的，但是后来他们也做了调整，而且他们具有本地的一些优势，对于他们来说，一下子有了好的参考，直接照搬就能够快速提高，我们在竞争中要掌握的核心竞争力便是如此，如果太容易被复制，或者是显而易见的问题，那么短期内改进空间很大，随着时间的推移，会带来更多的挑战。 

第二是水果的定位，对于一个水果店来说，需要考虑的因素确实很多很复杂，而且要考虑季节因素，根据经营情况不断调整。以下是一些水果的数据，看了以后发现确实有不少的门道。
