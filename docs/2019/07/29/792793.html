1.原有虚拟机jdk的卸载：
一般Vmware虚拟机安装后，会有自带的jdk- openjdk
查看命令：
rpm -qa | grep java

卸载命令：
rpm -e java-1.6.0-openjdk-1.6.0.41-1.13.13.1.el6_8.x86_64 tzdata-java-2016j-1.el6.noarch java-1.7.0-openjdk-1.7.0.131-2.6.9.0.el6_8.x86_64 --nodeps

其中： java-1.6.0-openjdk-1.6.0.41-1.13.13.1.el6_8.x86_64 java-1.6.0-openjdk-1.6.0.41-1.13.13.1.el6_8.x86_64 java-1.7.0-openjdk-1.7.0.131-2.6.9.0.el6_8.x86_64 --nodeps 是查询出来的三个jdk信息；

卸载完后，重新输入查看命令，什么也没有，就表示卸载干净了！

2.将新的jdk文件上传服务器，进行解压即可，解压命令：
tar -zxvf jdk-8u221-linux-x64.tar.gz -C /opt/xu/soft

环境配置：修改profile文件
vim /etc/profile
配置以下内容：
export JAVA_HOME=/opt/xu/soft/jdk1.8.0_221
export PATH=:$JAVA_HOME/bin:$PATH

配置完后，执行 source /etc/profile生效

最后：查看jdk信息,执行下面命令：
java -version
