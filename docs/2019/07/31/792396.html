DistutilsOptionError: must supply either home or prefix/exec-prefix -- not both

 

解决办法：

在/usr/local/lib/python2.7/site-packages目录下新建一个 setup.cfg 文件；

在文件中写入以下内容： 

[install]
prefix=

重新运行pip install 即可。
