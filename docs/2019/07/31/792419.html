量化交易是近期比较火热的项目，不管是币值涨跌，只要两个交易所之间币值有所差价就可进行套利。而在币值波动大的时候，套利的成功率会高很多。自动量化交易软件主要是让币种数量增加，但市值不是其能左右的。

自动交易软件系统主要是通过处理数字货币交易和监控全球数字货币市场价格变化，进行反复的高卖低买。 

这意味着投资者不需要做任何事情来监控市场，只需要设置机器人的参数即可开始自动网格交易。而且该交易机器人对用户完全透明，所有交易信号都根据用户设置自动打开。

量化交易机器人的主要包括以下类型 ：

一，直接与交易所进行交互（通常使用 API 获取和解释相关信息），并根据对市场数据的解析代表用户进行买卖交易。机器人做出这些决定，跟踪市场价格变动，并根据预定义和预编程的规则作出反应。

一般来说，尽管通常可以根据用户的口味和喜好对机器人进行相应编程，交易机器人还是会分析市场行为，如交易量、订单、价格和时间。

二，交易机器人在一个加密货币交易所进行交易，以较低的价格买入货币，再以较高的价格卖出，从而获得收入。

三，套利机器人，它们只在几个交易所交易，通过在汇率较低的交易所购买货币，在汇率较高的另一个交易所卖出，赚取利润。 尽管交易所之间的汇率差异现在小很多，但套利机器人仍然时不时地出现，这些交易机器人可以帮助用户充分利用这些汇率差异。
