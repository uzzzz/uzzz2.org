同一台电脑安装多个JDK

  • 柚子社区
  • Donate

1.在  https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html 中下载jdk

2.配置环境变量，例如

  JAVA_HOME6：F:\dev\Java\jdk1.6.0_27

 JAVA_HOME8：F:\dev\Java1.8\jdk1.8.0_101

 JAVA_HOME：%JAVA_HOME6% 或 %JAVA_HOME8%

 CLASSPATH：.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar（不要忘记前面的小点）

 在PATH后追加：;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

3.查看当前系统的JDK环境， cmd查看 java -version 。

4.如果当前的jdk环境没有生效，1）与2）都操作试试看

      1）删除  C:\Windows\System32目录下的与java相关的运行程序，列如java.exe，javax.exe，javac.exe（如果没有相关java程序请忽略）

    2）删除C:\Program Files (x86)\Common Files\Oracle\Java\javapath、C:\ProgramData\Oracle\Java\javapath 目录下的与java相关的运行程序，列如java.exe，javax.exe，javac.exe（如果没有相关java程序请忽略）

5.切换JAVA_HOME的变量，再次cmd 查看 。
