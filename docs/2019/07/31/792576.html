本人hadoop环境搭建：hadoop版本：CDH
CDH相关软件包下载地址：http://archive.cloudera.com/cdh5/cdh/5/

hadoop使用版本：hadoop-2.6.0-cdh5.15.1
hive使用版本:hive-1.1.0-cdh5.15.1

使用命令

wget http://archive.cloudera.com/cdh5/cdh/5/hadoop-2.6.0-cdh5.15.1.tar.gz


安装hadoop所必须要的条件
ssh、Java
本人Java采用1.8

这个命令可以很好的传输文件

scp 文件 root@192.168.0.1:~/

之后解压java

tar -zxvf jdk-8u9-linux-x86.tar.gz -C ~/app/


之后把jdk目录配置系统环境变量中

vi ~/.bash_profile


export JAVA_HOME=/home/root/app/jdk1.8.0_91
export PATH=$JAVA_HOME/bin:$PATH


之后进行

source ~/.bash_profile


如果看到这里，说明你的java环境配置成功

java -version


ssh需要配置
之前查看你的系统是否安装了ssh

ls -a
ls -la


本人之前是没有安装ssh的

ssh-keygen -t rsa



cat id_rsa.pub >> authorized_keys


修改一下权限

chmod 600 authorized_keys


之后登陆就不需要输入密码了

接下来正式安装hadoop

解压压缩包

 tar -zxvf hadoop-2.6.0-cdh5.15.1.tar.gz -C ~/app/


bin:hadoop客户端名单
etc:hadoop相关的配置文件存放目录
sbin:hadoop启动相关进程的脚本
example:内部一些依赖包
share:常用例子使用

之后进行hadoop配置

cd  etc/hadoop
vi hadoop.env.sh




配置core-site.xml

 <property>
        <name>fs.defaultFS</name>
        <value>hdfs://localhost:8020</value>
    </property>


配置hdfs-site.xml

<property>
        <name>dfs.replication</name>
        <value>1</value>
    </property>
<property>
	<name>hadoop.tmp.dir</name>
	<value>/home/root/app/tmp</value>
</property>


slaves(本人也只是单机安装的，如果是集群，需要配置节点对应ip)

配置hadoop环境变量

export HADOOP_HOME=/root/app/hadoop-2.6.0-cdh5.15.1
export PATH=$HADOOP_HOME/bin:$PATH

启动HDFS：第一次执行的时候一定要格式化文件系统

hdfs namenode -format



启动集群

cd ..
cd sbin
./start-dfs.sh


说明hadoop部署成功了
