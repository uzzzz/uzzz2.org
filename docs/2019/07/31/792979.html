错过了AWS技术峰会2019（上海站），所以7月31日的AWS技术峰会2019（北京站）说什么也不能再错过。早上8：30，北京国家会议中心，像往年的峰会一样人潮涌动。笔者的眼光不经意间扫过一个互动体验区，一位“奶奶级”的粉丝正玩得高兴，旁边的“吃瓜群众”也目不转睛。一场技术峰会让各方人马趋之若鹜，不可否认得益于AWS品牌的影响力，但更重要的还是云计算真正开始深入人心。



即刻构建你的云


不知别的参会者希望有何收获，反正我的关注点已经不仅仅在AWS又发布了什么新服务、新功能，而是AWS云又进入了哪个新的行业和领域，企业是如何使用AWS云服务并从中受益的。今天，上不上云已经不是问题，如何上云才是重点。AWS大力倡导的“即刻构建”，正好是这一趋势的最好诠释和反映。


AWS全球技术与开发者布道师团队总监Ian Massingham的主题演讲，表面看是在介绍AWS的产品与服务，实际上都是为引出云客户的成功经验做铺垫。阿迪达斯、外研在线、猎豹等各细分行业内的头部客户的上云、用云体验是最大的亮点。


AWS刚刚宣布了与复旦大学附属华山医院合作的最新进展。复旦大学附属华山医院成功将预约总线、定量药理分析、一期临床科研等应用迁移上云，部署在由光环新网运营的AWS中国（北京）区域，在加速新应用部署、降低基础设施成本的同时，从医教研等多方面提升了医院的综合实力和水平。举例来说，利用Amazon Elastic Compute Cloud (Amazon EC2)，华山医院发布了预约总线系统，患者可以通过各种渠道预约专家，预约程序通过智能筛选实现精准预约，减少了患者去医院的次数和等待时间，让每年超过110万人次的患者受益。此外，在科研方面，华山医院利用AWS超强的计算资源进行定量药理分析，高效、低成本地解决了大量计算的问题，成功开发了万古霉素的个性化用药决策辅助系统“SmartDose”，并成功服务80余家医疗机构。AWS的稳定可靠、先进技术和丰富的产品功能不仅是复旦大学附属华山医院一家将AWS作为首选云服务的原因，也是AWS客户共同的心声。


作为一家中国企业，猎豹全年收入的60%来自海外，像许许多多“出海”的企业一样，拥有一家业务覆盖全球的可靠的云服务商是多么必要。猎豹现在又要将业务拓展到AI和机器人领域，而机器学习正是AWS的拿手强项。曾经在亚马逊工作过的外研在线CEO、董事商其坤对AWS“客户至尚”的理念十分认同。每当业务遇到问题，与外研在线对接的AWS专属行业顾问便会提供点到点的支持，不是远程支持，而是深入到外研在线的技术团队中。按商其坤的话说，双方的合作产生的是一种化学反应。让商其坤印象深刻的是，针对外研在线的业务和存在的问题，AWS总是能给出前瞻性的建议，并分享其他客户的成功案例和经验，主动帮助外研在线降低成本、提升效率。


AWS全球副总裁、大中华区执行董事张文翊


刚上任不久的AWS全球副总裁、大中华区执行董事张文翊表示，当前，越来越多的公共服务机构希望摆脱传统IT基础架构的束缚，创建云环境，从而获得更多自由和控制，实现创新和扩展。AWS正与各行各业的客户一起打造数字化转型的成功样板。


AWS要跑起来，还要跑得更快


2013年12月18日，AWS宣布正式进入中国，至今5年多的时间，AWS完成了在中国从0到1的第一步跨越，AWS“前店后厂”的模式在中国落地。张文翊上任后，应该说AWS在中国发展的新阶段也从此展开。在中国市场上，AWS要走得更快，或者说要跑得更快。面对大规模企业级客户的上云挑战，AWS要做更多细节上的工作，在快速的技术创新与云的稳妥落地之间找到一个更好的平衡点。


在形形色色的公有云、公有云服务商不断涌现之后，AWS依旧显得那么“特别”，先行一步、长期积累固然是AWS的天生优势，张文翊总结出的五大优势更让人信服。AWS是这样一朵云：技术领导者的云、注重安全和隐私的云、覆盖全球的云、全球最佳商业实践的云、Day 1赋能行业Builder的云。


举例来说，AWS坚持创新，不断丰富功能，2018发布的新功能高达1957项。AWS的创新遵循客户至尚的理念。AWS有一项Trusted Advisor的功能，会监测客户对AWS资源的利用情况。如果资源使用率过低或者空转，AWS会主动联系客户，让客户省下这笔钱。在过去几年中，通过AWS Trusted Advisor，每年让客户少花费数亿美元。


AWS全球技术与开发者布道师团队总监Ian Massingham


就在峰会举行前的一天，AWS中东（巴林）区域正好开通。目前，AWS在全球22个地理区域内运营着69个可用区，同时计划增加开普敦、雅加达和米兰三个区域。


张文翊表示，AWS从Day 1就坚持赋能行业，依靠合作伙伴，形成了强大的合作伙伴生态。云计算一定要分工合作。AWS的核心能力是技术和产品创新能力，AWS是云服务的Builder，而合作伙伴是行业应用的Builder。AWS与合作伙伴扬长补短，集合整个生态的力量，可以更好地服务客户。


我们人人都可以成为云的构建者，当然更是云的使用者和受益者。AWS倡导的“即刻构建”，其中一层重要的含义是，现在就是拥抱云计算的最佳时机，企业用户应该马上行动起来，包括AWS在内的云服务商已经准备好了，可以提供全面可靠、切实可行的上云支撑。商其坤说的好，在任何时候都不要做与大势相违背的事情，而应该顺势而为，尽早拥抱云计算。


持续创新是常态


既然是技术大会，还是要说说AWS在技术和服务上的创新。


AWS技术峰会2019（北京站）设立了10大技术分论坛，涵盖近70个技术话题，全面展示了AWS在人工智能、机器学习、数据库、大数据分析、物联网、安全等方面的最新进展和解决方案。


2018年，AWS中国（北京）区域和AWS中国（宁夏）区域推出了数百项新服务和新功能。2019年，创新仍在提速。


Amazon EC2现已在由光环新网运营的AWS中国（北京）区域和由西云数据运营的AWS中国（宁夏）区域推出下一代通用型实例M5，通用型可突增实例T3以及一种新类型的R5内存优化型实例。


AWS技术峰会2019（北京站）现场


M5和T3实例提供Intel Xeon Platinum处理器（之前的代号为Skylake），而M5a和R5a实例提供最新的AMD EPYC处理器。新的EC2实例类型采用Amazon EC2 Nitro系统，该系统结合专用硬件和轻量级管理程序，旨在提供与裸机媲美的性能。M5 实例具备平衡的计算、内存、存储和网络资源，与M4实例相比，性价比提高47%。T3 实例的CPU性能可以突增，与T2实例相比，性价比提高了 30%。M5a、T3a及R5a价格与M5、T3及R5相比，均下降10%。


AWS IoT Greengrass现已向由光环新网运营的AWS中国（北京）区域的客户提供。AWS IoT Greengrass可将AWS服务无缝扩展至边缘设备，因此可以在本地操作其生成的数据，同时仍可将云用于管理、分析和持久存储。借助AWS IoT Greengrass，连接的设备可以运行AWS Lambda函数、基于机器学习模型执行预测，保持设备数据同步以及与其他设备安全通信，甚至在没有连接互联网的情况下也可实现这些功能。


Ian Massingham表示，物联网、机器学习以及数据库应用既是应用的热点，也是AWS的业务重点。即刻构建，AWS要为企业客户提供全面的上云服务和支撑。
