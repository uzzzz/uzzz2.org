90 后程序员薪资大揭秘：有人刚毕业年薪 200 万，有人月薪不足 1 万

作者 | 素年清时

责编 | 伍杏玲

出品 | 程序人生（ID：coder_life） 

相信大家最近都被华为给几名新员工开出的巨额工资给刷屏了，根据任正非签发的一份总裁办电子邮件，华为对八名 2019 届顶尖学生给出百万高薪，其中，有两名员工的年薪高达 182 万- 201 万元。

这八名员工均为博士学历，按正常入学年龄和升学情况算，八名博士的出生年岁应该都在 90 年左右。

一毕业就年薪过百万的 90 后程序员毕竟是IT界中的翘楚，但大多数 90 后程序员的工资是很难达到这个数目的。本文采访了几位 90 后程序员（媛），谈谈他们的薪资情况。



李克，男，91 年，坐标北京

说起来，我也是根正苗红、纯科班出身的程序员了，然而与这8位华为大神比起来，我真是严重拖了程序员的后腿。

我 2017 年硕士毕业于北京某 985 高校，计算机相关专业，入职某一线互联网公司，第一年年薪 18 w，年终奖另算。这样的薪资并不算高，据我知道的，在我们那一届毕业的同学里，有好几个拿到了40多万的Offer。人比人，没法比。

因为在大厂，里面的工作氛围挺不错的，同事们都很厉害。里面有不少技术大神，人都挺nice的，有时候遇到问题请教他们，只要有时间，他们都会耐心地给你讲解清楚。当然，大多数情况下，他们都是没时间的，大家都很忙。

刚入职的那几个月，我从来没在晚上十点前下过班，加班到半夜打出租车回去，公司报销车费。

有付出就有回报，在这里成长得非常快，技术水平一点一点地积累，薪酬也会一点一点地涨上去。现在我的月薪15k，年终奖根据KPI而定，但是不会少于1个月的工资数。

 


大白，女，90年，坐标南京

生于忧患，死于安乐。我选择了后者。

大概是由于早年学业精进的缘故吧，我的工作比较顺利，起码令我感到非常满意。

15年硕士毕业，通过校招进入一家国有企业，一干就是4年，不出意外的话，我还将继续干下去。薪水和工作内容都挺好的，我为什么要走呢？

跟刷爆朋友圈的华为应届博士的百万年薪没法比，我的年薪刚刚十万出头，不过国企嘛，大家都懂的，公积金缴纳的多，以各种名目下发的奖金和福利也多。最重要的是，在这里工作轻松，朝九晚六按时下班。另一方面，这份工作也稳定，旱涝保收，在过去的一场场裁员风波里，我认识的好几个同学都没能幸免，还好我高枕无忧。

人在舒适的圈子里待久了就会变懒。朋友曾给我介绍对象，一个在上海工作的程序员，人挺有上进心的，挣得也比我多，跟他一比，我觉得自己简直浪费我的专业和学历。不愿降薪来南京发展的他劝我去上海，因为去了上海准加薪。

权衡一番，我还是放弃了，放弃了这个男友和上海，我舍不得放弃现在的生活和工作环境。

就让我“死”于安乐吧！起码我享受过安乐。

 


阿俊，男，91 年，坐标深圳

我在 2013 年毕业于南京某双非普通一本院校，毕业后去了一家小外包公司，软件大道那边遍布着许多家这种类型的外包公司，这是南京的一个特色。工资不高，每个月到手才五千多一点。

一年多之后，经大学一哥们介绍辞职去了深圳，一线城市，发展机会多一些。

因为有哥们帮忙，一去深圳就落实了工作，薪水较之前翻了一番不止。

深圳气候炎热潮湿，刚去的那段时间并不适应，膝盖以下部位出了好多疹子，去药店买了药膏回来涂也无济于事。

但就像高架桥底下疯长的绿色植物和出租屋里怎么也消灭不了的小强一样，这座城市之中的生物有着一股顽强的生命力，连人也是如此。

腿上的疹子因为瘙痒难耐被我一个一个地抓破，流脓流水，再结痂愈合。没有挨不过去的伤，因为我想留在这儿。

一晃几年过去了，我的薪水经过几次增长，现在一年三十万左右。在深圳，这样的收入再普通不过了。每天上下班坐地铁，在每一节拥挤的车厢里，就有一大打拿着跟我差不多年收入的 90 后程序员。

 


杨旸，男，92年，坐标昆山

跟华为招收的那几个应届博士生比起来，我真是弱爆了。我既没有他们的名校、博士学历，也进不了华为这样的大公司。

几年前，我二本毕业，非计算机专业。第一份工作是去车间做维修机工，白班夜班两班倒，工资低得要命。当时想着慢慢熬呗，熬出了头就好了。

我总算没把这份工作熬到头。机缘巧合之下，报了个IT培训班，转行改做码农。几个月集中学习，毕业后被推荐到一家外包公司面试，面试过程就是走个过场，我毫无悬念地被录用了，当时定的薪资是6k每月。还行，跟第一份工作相比，有了那么一点的涨幅。何况，HR跟我说每年都会根据个人的表现调整薪水，而且每个项目结束都有项目奖。

入职之后才发现，HR的话拿来当做口号听一听就好了，千万不要当真。

首先，每年的确会调薪，却是大锅饭似的，人人有份。原因很简单，我们的工作形式属于人力外包，公司根本看不到你的具体表现，只能根据甲方公司的反馈作为考核依据。而我们在甲方干活，只要不出大纰漏，甲方不会给予差评，当然也不会给予特大好评，都是统一样式的好评。所以最后的结果，几乎所有人的薪水都会每年上调一千。至于项目奖金，由于其复杂的换算方式和冗长的发放周期，反正我到现在领到的项目奖金不超过一个月的工资。

我深知这样的工作环境并非长久之计，所以期间我一直抽时间自学，看书看视频，跟着网上的项目写代码，为早日跳到下一个平台努力地付出着。

断断续续地自学了近一年（原谅我笨）之后，请了年假，跟领导说是出去浪，其实是出去看看机会了。7天面了十几家，最后铩羽而归，败得一塌糊涂。

原来以为照着之前互联网大火的形势，攒点经验、学点牛掰的技术框架，总能拿个像样的Offer回来。然而，大环境并非我这样的等屌丝所能预料到的，互联网“寒冬”突然就来了，差点冻死了我这只菜鸟。

我该庆幸寒冬没有波及到我所在的外包公司，起码我没听说哪个同事被裁，在这里，起码我们是安全的。

现在的月薪8k，好在项目驻场实施有出差补贴和租房补贴，加起来也有一万。日子凑合着过吧。

有人住高楼，有人在深沟，有人光万丈，有人一身锈。我们无法成为他人，我们仅能成为我们自己。

正如这句话所说：这个世界非常多样，自我实现的方式也绝不止哪一种。我们每个人都不一样，永远不要试图成为他人，那是妄念。我们唯一能做的，就是看清自己并努力成为更好的自己，这条路永远都不会错。
