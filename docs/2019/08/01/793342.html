4、编写第一个程序

package main

import "fmt"

func main() {
	var name string
	fmt.Println("请输入你的名字:")
	fmt.Scanln(&name)
	fmt.Println("你好", name)
}

编写上面的代码保存为hello.go,进入hello.go文件所在目录运行命令go run hello.go

5、编译可执行exe程序

go build hello.go

执行上面的命令在windows下会生成一个exe，运行exe与运行其他可执行程序一样
