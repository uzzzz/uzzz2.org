炎炎夏日，DB-Engines 的8月榜单已经发布，本月积分 MySQL 获得了最显著的增长，较上月增加了 24分，Oracle 获得了 18 分的增长，Oracle 公司的两个王牌产品，闪耀 8月。

在前10名中，除了 Oracle 和 MySQL 之外，其余产品的波动不大，但是前10名中有 6 个产品出现下跌，我们将此归结为季节性因素，历年夏日总是一个蛰伏的时期。


从前三名的趋势上看，分别在2019年年初跌至低谷，随后获得超过六个月的持续增长，传统数据库有望重新确立增长趋势，摆脱自2016年开始的持续下滑。


从中国市场来看，MySQL 的应用正在向纵深发展，其热度不减。例如工商银行曾经在今年的数据库大会上分享其 MySQL 转型经验，大量的 MySQL 开始支持金融级业务应用。

此外，虽然公开的信息并不多，但是中国电信基于 MySQL 的增强 TeleDB 已经能够为电信及其合作伙伴提供基于交易或订单系统的云数据库服务。

虽然用 Oracle公司 的一个开源产品（MySQL）去替代 Oracle公司 的一个商业产品（Oracle）略显分裂，但是无疑，很多企业已经走在了大踏步前进的道路上。


而对于 Oracle公司 来说，这种矛盾还会长期交织在一起，一边是闭源、一边是开源；一边是收费，一边是暂时免费。好在，这两者将会长期盘踞在排行榜上不去，Oracle始终是数据库的王者。
