大学毕业总结

  • 柚子社区
  • Donate

      四年前的今天，我拿到了大学录取通知书，打开通知书，看着上面专业名字（信息与计算科学）一脸迷茫的感觉我到现在依然记忆犹新：‘’怎么会是跟数学相关的专业。。。“，当想着算了数学专业就数学专业吧，只要认真学应该也还行。抱着这样的心理暑假时间过得飞快，在开学第一堂课上，我才知道信息与计算科学后面还有一个括号写着（软件工程），那时，我才知道我的专业未来是要当程序员的，对于电脑不熟，打字很慢的我来说，这个信息量有点大，无异于平地一声雷，头很大。
        几乎每门编程语言都是从hello world开始的，诚然，我也是。大学的第一个实验作业就是，在自己的电脑上配置好Java代码运行所需要的环境，尽管现在看来老师当时的文档指导写得非常详细了，但是对于连 public static class这些是啥意思的我来讲，看这些代码还有一堆跟如何配置环境变量有关的描述，就跟看天书一样，一脸懵逼，于是，要交实验的前一天晚上，整个宿舍自然而然的熬夜了，从晚饭吃完五点多到深夜两三点就为了截一张从命令行窗口运行出来的hello  world。虽然现在回头想想当时的自己很傻比，但hello world 成功运行出来的时候的那份兴奋与感动，我想我一辈子都忘不了。

      对于编程，从排斥到接受到尝试再到慢慢喜欢，在这个过程中有辛苦也有解决问题的欣喜与欣慰，也许就是解决问题的成就感：做出人生中第一个数据库管理到书店APP再到校园图书馆网站，让我逐渐相信，对于编程只要我想学，想研究，没有什么是实现不了的。

     转眼间，四年的大学时光已经成为过去式了，苦熬的毕业设计、五六月份的毕业答辩，毕业照。毕业典礼，于现在而言就想梦一样，经历过，但很虚无，如果不是有朋友圈的照片，感觉就想上辈子的事情了一样。庆幸的是自己这一路走来，放纵过，努力过，不曾后悔。

     小时候渴望长大，渴望变成大人的模样，渴望自由自在，独当一面不受约束。我曾天真的以为，只要努力读书，考上好的大学，我的人生就是完满的了，就不需要烦恼了。到去年开始出来找工作实习的时候，才惊觉，人生才刚刚开始。越长大肩上的重量一直在递增，而我们只能负重前行，这是一个人长大，成长的必经过程。

      而已经毕业的我，已经正式成为社会人的我，该怎么做才能把接下来的路一步步地走对呢？这个问题只能一边前进一边慢慢确定，因为往事不可追，未来无法预测，我能做的就是珍惜当下，去体验，去承担，去成长。

      人无法改变自己的出生，无法改变周围的人和事，无法避免会降临在自己身上的事，那么就改变自己吧，让自己慢慢成长，慢慢变得更好，不能改变那就去适应，只要我们知道自己要的是什么，知道自己是谁，那么我们披着什么样的外衣与面具又有什么关系呢?

      2019年还剩几个月，我的目标是:

      1、认真工作努力提高自己的业务水平；

       2、在工作之余多看书把自己的基础打扎实了；

       3、坚持防晒，努力变白，多锻炼身体；

       4、努力攒钱，做个快乐的开心果；

      flag立在这里了，不许偷懒哈！！！！！
