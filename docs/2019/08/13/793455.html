1. 探索属性

在做数据分析之前，我们需要了解我们的数据，那么如何查看数据的属性，操作如下：

a=np.arange(8)

print(a)						# 打印数组
print(a.ndim)					# 秩 1
print(a.size)					# 整个数组中元素个数8个
print(a.shape)					# 结果 (8,)
print(a.dtype.name)				# 数组中数据的类型 int64
print(type(a))					# 数组类型 <class 'numpy.ndarray'>
print(a.itemsize)				# 数据类型占用的空间大小 8

2. 探索接口

  • 最大元素，最小元素
a = np.array([[32, 15, 6, 9, 14], 
              [12, 10, 5, 23, 1],
              [2, 16, 13, 40, 37]])

print(a.min())				# 所有元素中最小的 1
print(a.max())				# 所有元素中最大的 40
  • 求和，累加

（1）全部元素求和

# 无论数组是几维，求解所有元素的总和
print(a.sum())		# 235

（2）按行或列求和

# 因为数组是多维的，所以可以在特定的轴上，使用数组的方法
# axis=0是按照列方向进行函数操作；axis=1是按照行方向进行函数操作
print(a.sum(axis=0))		# 按照列方向进行加和 [46 41 24 72 52]

（3）按行或列累加

# 按照轴方向进行累加
print(a.cumsum(axis=1))
------------------------------------
# 结果为第一列不变，第二列为第一列+第二列，第三列为第二列+第三列。。。以此类推
[[ 32  47  53  62  76]
 [ 12  22  27  50  51]
 [  2  18  31  71 108]]
  • 排序：np.sort()

（1）与随机序列一起使用，排序后用作在数轴上

# 得到一个从小到大排列的，呈正态分布的，一维数组
a=np.random.normal(10,2,50)
np.sort(a)

# 当是一个多维数组时，使用axis=0是按照列方向进行函数操作；axis=1是按照行方向进行函数操作
a=np.random.normal(10,3,(2,4))
print(a)
b=np.sort(a,axis=0)
print(b)

-------------------------

heights = np.array([49.7, 46.9, 62, 47.2, 47, 48.3, 48.7])

  • 求平均值：np.mean()

(1) 求数组的平均值

# 求一维数组的平均值
a = np.random.normal(10,2,50)
np.mean(a)					# 10.309453780901238 之所以不是10，是因为计算的是随机数组

# 求二维数组的平均数，使用axis=0是按照列方向进行函数操作；axis=1是按照行方向进行函数操作
ring_toss = np.array([[1, 0, 0], 
                       [0, 0, 1], 
                       [1, 0, 1]])
                       
np.mean(ring_toss) 						# 所有元素的平均数 0.44444444444444442
2. np.mean(ring_toss, axis=0)			# 以列为单位，计算每列的平均数 array([ 0.66666667, 0. , 0.66666667])

（2）求某一条件下数据的数量占总样本数量的百分比／概率

# 实际上，就是计算满足逻辑语句的值的百分比
class_year = np.array([1967, 1949, 2004, 1997, 1953, 1950, 1958, 1974, 1987, 2006, 2013, 1978, 1951, 1998, 1996, 1952, 2005, 2007, 2003, 1955, 1963, 1978, 2001, 2012, 2014, 1948, 1970, 2011, 1962, 1966, 1978, 1988, 2006, 1971, 1994, 1978, 1977, 1960, 2008, 1965, 1990, 2011, 1962, 1995, 2004, 1991, 1952, 2013, 1983, 1955, 1957, 1947, 1994, 1978, 1957, 2016, 1969, 1996, 1958, 1994, 1958, 2008, 1988, 1977, 1991, 1997, 2009, 1976, 1999, 1975, 1949, 1985, 2001, 1952, 1953, 1949, 2015, 2006, 1996, 2015, 2009, 1949, 2004, 2010, 2011, 2001, 1998, 1967, 1994, 1966, 1994, 1986, 1963, 1954, 1963, 1987, 1992, 2008, 1979, 1987])
millennials=np.mean(class_year > 2005)
print(millennials) 				# 0.2 即出生在2005年之后的人占总人数的20%

a = np.random.normal(10,2,50)
np.mean(a > 11)					# 0.4 即随机数中，大于11的数值比例／从随机数字中去一个数，大于11的概率
  • 求标准差：np.std()
a = np.random.normal(10,2,50)
np.std(a)				# 1.7086488749575695 之所以不是2，是因为计算的是随机数组
  • 求中位数：np.median()
my_array = np.array([50, 38, 291, 59, 14])
np.median(my_array)				# 50.0
  • 求某个百分位上的数值：np.percentile(数组名，百分比)
d = np.array([1, 2, 3, 4, 4, 4, 6, 6, 7,  8, 8])
np.percentile(d, 40)			# 4.00
