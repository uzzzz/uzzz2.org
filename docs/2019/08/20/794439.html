8.Docker搭建testlink

  • 柚子社区
  • Donate

镜像获取

  • 拉取镜像

docker pull bitnami/mariadb:lasted

docker pull bitnami/testlink:latest

启动容器

启动容器，首先启动mariadb,后面启动testlink

  • mariadb

docker run -d --name mariadb -e ALLOW_EMPTY_PASSWORD=yes -e MARIADB_USER=bn_testlink -e MARIADB_DATABASE=bitnami_testlink -v E:/Docker/testlink:/var/lib/mysql -p 8088:3306 bitnami/mariadb:lasted

  • testlink

docker run -d --name testlink -p 8099:80 -p 444:443 -e ALLOW_EMPTY_PASSWORD=yes -e TESTLINK_DATABASE_USER=bn_testlink -e TESTLINK_DATABASE_NAME=bitnami_testlink --link mariadb bitnami/testlink:latest

访问容器

在访问前先检查容器是否启动

docker ps（目前启动的容器）

docker ps -s -a（显示过去启动的容器）

使用docker logs -f testlink查看testlink容器的log信息

  • 浏览器中访问

http://localhost:8099/

用户名：user
密码：bitnami
