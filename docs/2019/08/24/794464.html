一、准备工作

安装hive的前提是已经在主机中安装好JDK、hadoop、zookeeper如果准备还没有安装这样三个，可以参考如下文章安装

JDK:https://blog.csdn.net/qq_42668255/article/details/97368887

hadoop:https://blog.csdn.net/qq_42668255/article/details/91385098

zookeeper:https://blog.csdn.net/qq_42668255/article/details/97483170

二、下载

官方网址：https://mirrors.cnnic.cn/apache/hbase/

本人云盘资源：https://pan.baidu.com/s/1ByjyHMUmMcSn1oGm9AIVVg

提取码：n7zq

三、安装

  1. 解压 tar -zxvf hbase-1.4.10-bin.tar.gz
  2. 环境配置变量
    1. export HBASE_HOME=/home/ubuntu/hbase-1.4.10
    2. export PATH=$HBASE_HOME/bin:$HBASE_HOME/conf:$PATH
    3. source  /etc/profile
  3. 修改hbase-2.0.0/conf/       vim hbase-env.sh
    1. export JAVA_HOME=/home/ubuntu/jdk1.8.0_211 
    2. export HBASE_CLASSPATH=/home/ubuntu/hbase-1.4.10
    3.  export HBASE_MANAGES_ZK=false (使用外部zookeeper需要加入此行代码否则回报 2181端口被占用) 
  4. hbase-site.xml    vim hbase-site.xml
  5. <property>
    <name>hbase.rootdir</name>     //hdfs中的文件目录
	<value>hdfs://master:9000/hbase</value>
</property>
<property>
	<name>hbase.zookeeper.property.dataDir</name>  
	<value>/home/ubuntu/zk35</value>
</property>
<property>
	<name>hbase.master</name>  
	<value>master:60000</value>
</property>
<property>
	<name>hbase.tmp.dir</name>  
	<value>/home/ubuntu/hbase-1.4.10/hbase-temp</value>
</property>
<property>
    <name>hbase.cluster.distributed</name>
    <value>true</value>
</property>
<property>
    <name>hbase.zookeeper.quorum</name>
    <value>master</value>
</property>
<property>
    <name>dfs.replication</name>
    <value>1</value>
</property>
  6. 启动hbase:   start-hbase.sh 
  7. 进入shell窗口：hbase  shell
