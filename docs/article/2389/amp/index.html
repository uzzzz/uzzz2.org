Pycharm的安装过程以及主题颜色、字体颜色格式的修改

fandyvon
3 years ago

（1）pycharm的安装

Pycharm下载地址：https://www.jetbrains.com/pycharm/download/#section=windows

在官网下载完成后

下一步

启动Pycharm，勾选do not import settings 不导入任何配置

调整使用主题的颜色【暗色调|明亮色调】

对于新手~罗列的每个插件，都不安装，直接启动Pycharm即可【点击start using PyCharm按钮】

JetBrains Account: 使用Jetbrains账号+密码激活

Activation Code：激活码激活

License Server：注册服务器激活~输入http://16个字符，点击Activate激活即可。

（2）主题、字体的修改

Create New Project：创建一个新项目

Open：打开一个存在的文件夹~当成项目文件夹

Check Out From Version…：协同开发，从远程服务器克隆项目到本地。

工具界面

常见配置选项：

1. 配置工具展示字体大小、编辑器、主题

打开设置对话框

Appearance 外观 ，调整编辑器所有字体和大小

Editor编辑器

              Font 字体

右侧窗口

Font：调整编辑器使用字体，推荐Source Code Pro

Size：字体大小，~开发建议16号字体，讲课~建议28号字体

LineSpace：行间距默认1.0~推荐1.2

打开系统鼠标滚轮控制字体大小选项【Editor 编辑器  、General 常规的】

 3. 配置主题颜色

调整软件 主题 【明亮、暗黑】

调整代码编辑区颜色~如果和软件主题不一致，会有一个弹出窗口提醒是否同步【选择No】

4.调整编辑区背景图片

5.查看软件的注册信息

6.运行Python代码

如果你的Pycharm中运行不了Python代码，按照下面的方式，配置Python运行环境

Project：项目

Interpreter：解释器

New Environment 新创建一个环境

Existing Environment 使用已经存在的环境(系统中安装好的python)
