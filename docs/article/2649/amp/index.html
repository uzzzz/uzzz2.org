4. padding

之前的做法优缺点“

1. 会使得图片矩阵变小

2. 在矩阵中间的点比边缘的点使用更多，边缘的信息丢失

所以在使用过滤器之前，可以加边缘（padding），每个边都加一个像素，就能削弱上面两个缺点

填充的选择：

Valid ：no padding

Same: 输入输出（过滤后）的大小相同，使得n+2p-f+1=n，p=(f-1)/2

其中，n是输入的维度（n * n)，p是padding，f是过滤器的维度

f一般是奇数，方便填充，会有中心点。
