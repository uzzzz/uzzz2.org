一:设备管理器介绍：

Android 2.2 SDK提供了一个可管理和操作设备的API叫DevicePolicyManager，

使用这个API你可以接管手机的应用权限，

对手机做出很多大胆的操作，

比如锁屏、恢复出厂设置、设置密码、强制清除密码，修改密码、设置屏幕灯光渐暗时间间隔等操作。

	    }

	        return "警告禁止用户的请求";
	    }

	        showToast(context, "设备管理器关闭"));
	    }

	    @Override

	        showToast(context, "密码改变"));
	    }
	
	}
