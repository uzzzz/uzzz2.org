tor 安装及使用教程

2015/2/27 Uncategorized




最近研究深层网络，需要用到TOR，因众所周知的原因，网上的TOR教程大多过时了。好在TOR发布了新的TOR4.0，经试用，完全可以满足基本浏览要求。

   1、  下载TOR软件

软件官网下载地址是 https://www.torproject.org/projects/torbrowser.html.en，可选择中文版下载  （当然要×××才能访问）

或者找到网友分享的下载,如，http://pan.baidu.com/s/1qWNTQjU

3、结合其它软件使用方法

TOR Browser 内置了 TOR。这个内置的 TOR 会在本机地址（127.0.0.1）开启一个 SOCKS 端口，端口号是 9150

因此，如果某个网络软件跟 TOR Browser 安装在同一个操作系统中，并且该软件支持 SOCKS 代理，那么该软件就可以利用 TOR Browser 内置的 TOR进行工作。

例1：IE浏览 + TOR

假如你不喜欢 TOR Browser 内置的 Firefox，喜欢用自己的 IE。那么你可以设置 IE，让 IE走 TOR 提供的 SOCKS 代理。
