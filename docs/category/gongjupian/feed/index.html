一、设置 tor 代理

  1. 下载核心源码：
  2. 运行目录下的 Start Tor Browser
  3. 等待安装结束自动跳出浏览器
  4. 点击TorBrowser –> Add-ons
  5. 点击 扩展 ==> 在TorButton处点击选项
  6. 选择使用用户代理

到这里tor配置就结束了，之后配置浏览器使用该代理即可。
