使用Firefox+Tor+FoxyProxy+Noscript+IETab进行安全自由的网上冲浪

在网上看到有人用上面的组合上网，有时在查资料可能用到：

1.Firefox这个不多说了，现今自由的后生可畏的第二大浏览器。下载地址：http://www.mozilla.org.cn/

2.Tor这个可能有很多网友没听说过，他是一个很牛逼的代理软件：tor的全写是“The Onion Router”，是洋葱路由器的意思。它是一种点对点的代理软件，依靠网络上的众多电脑运行的tor服务来提供代理。tor代理网络是自动连接并随机安排访问链路的，这样就没有了固定的代理服务器，也不需要去费劲寻找代理服务器地址了。而且Tor的代理一般在2-5层左右，加密程度也比较3高。下载地址：http://tor.eff.org/（可能访问不到）。

3.FoxyProxy是Firefox的一个插件，用于和Tor沟通，通过设置以方便自动的区分需要使用代理的网站和不需要使用代理的网站。

4.Noscript也是Firefox下的一个插件，用于屏蔽网页上可能寄生木马病毒的脚本，你可以方便的针对每一个网站进行个性化的安全设置。

 

5.IE Tab也是Firefox下的一个插件，我们知道现在很多的网站都是针对IE开发的，尤其国内的很多网上银行都需要使用IE，而在firefox下根本无法运行，所以我们需要这个小插件，它可以在不需要启动Ie的情况下只需点击一下方便的将现有网页在firefox调用IE引擎重新解析一遍。以上三个插件下载地址： firefox插件网 搜索下载即可，大家需要注意的是插件下载页面是英文的。
