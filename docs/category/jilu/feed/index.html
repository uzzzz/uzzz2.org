前言

今天快播王欣的马桶MT、今日头条张一鸣的快闪、锤子罗永浩的聊天宝三款社交app上线，下载马桶简单的测试了一下。

介绍

APP使用说明定位为人脉暗网，影子朋友圈。

说明

  1. 需要使用手机号。
  2. 据说服务器已炸，之前一直提示“验证码错误”。
  3. 下午可用，但基本什么好的群聊或者也找不到什么群聊。
  4. 不看好，稀烂。

添加话题

  • 点击 左上角 “+” 创建群聊

  • 输入疑问或想打听的信息

  • 红包设置

  • 时效

群聊的时效是一小时，到期自动关闭。

  • 邀请
  1. 可点击右上角 “…” 通过微信等各种渠道邀请好友参与群聊。

  2. 通过话题邀请按钮进行邀请，如下图。
  • 隐匿性
  1. 不显示头像和昵称，通过扑克牌显示。
  2. 通过点击头像可查看对方加入的话题。
