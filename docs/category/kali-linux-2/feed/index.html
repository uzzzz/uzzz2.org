如何在Kali Linux 中安装和设置Tor

第一步：安装Tor

   

apt-get install tor

第二步：下载和设置Tor Bundle

前往https://www.torproject.org/projects/torbrowser.html.en下载适合你的系统的Tor Bundle

然后，进入你的下载目录并解压

cd ~/Downloads/
<pre name="code" class="plain">tar -xvJf tor-browser-*


 解压完成后，运行Tor Bundle

  




cd tor-browser*
./start-tor-browser.desktop


Tor 不能以root运行，会弹出错误：Tor Browser Should Not Be Run As Root, Exiting.

  
 解决方法如下： 


1.进入tor-browser*目录下的Browser目录，编辑start-tor-browser文件




cd Browser
vi start-tor-browser


 输入斜杠查找root 


/root


 找到如下代码： 


if [ "`id -u`" -eq 0 ]; then
	complain "The Tor Browser Bundle should not be run as root.  Exiting."
	exit 1
fi


 注释之 


2.cd到上层目录，运行tor-bundle




cd ..
./start-tor-browser.desktop


出现如下或类似错误：




执行




chown -R root *


 再次运行


./start-tor-browser.desktop



  


成功运行.
