Nyx是Tor的命令行监视器

安装 nyx

brew install nyx

  • 查看nyx版本
    nyx --version
    nyx version 2.0.4 (released November 5, 2017)

  • 运行报错
    nyx
    Unable to connect to tor. Are you sure it's running?
