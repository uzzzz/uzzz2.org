Axure RP 是一套专门为网站或应用程序所设计的快速原型设计工具，可以让应用网站策划人员或网站功能界面设计师更加快速方便的建立Web AP 和Website的线框图、流程图、原型和规格。

Axure RP 9拥有一个新的硬件加速渲染引擎，一个旨在加快保存和加载的文件结构，以及一个便于平滑缩放和更快速编辑的流线型画布。因此，您将获得比Axure RP所熟知的所有原型功能和文档功能更好的功能。

>>> AxureRP v9.0.0.3653最新版下载

根据客户的反馈，新版本Axure RP 9中Axure RP’s UI、内联编辑、交互构建等已经过全面重新设计和优化。其中，每一个更改都将使您的工作流程更轻松、更快速。今天，我们主要介绍一个最受欢迎的增强功能——Dark UI模式，以补充最新的macOS和Windows操作系统上的暗模式。

操作方法：

1、打开Axure RP的应用程序Preferences menu:
>在Windows上， File > Preferences
>在Mac上，Axure RP 9 Beta > Preferences
2、在“ General”标签的“ Appearance”下拉列表中选择“ Dark Mode ”。Mac用户可以选择使用系统设置让Axure RP与操作系统的UI设置相匹配。
