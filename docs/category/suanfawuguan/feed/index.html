安装

1.官网下载

不会翻墙的同学直接移步方法2，官网已被墙。
直接去官网上下载软件包tor-browser-linux64-5.0.6_zh-CN.tar.xz
https://www.torproject.org/download
不会安装的同学可以学习一下（高手绕道）

解压
（不过话说直接上网问的那些人：你们为什么不用man tar）
可以在终端中直接输入如下命令解压

tar xvJf XXX.tar.xz

根据dingdingdingchris的说法（我没试过）
对于tar版本大于1.22（http://www.gnu.org/software/tar/），可以直接用命令

tar -xvf XXX.tar.xz

因为从1.15起tar就可以自动识别压缩的格式。

不过在解压的时候我用xz命令解压遇到了报错

xz -d XXX.tar.xz
xz 输入文件有多个硬链接,跳过

好像重启了一下就好了……我也不知道有什么问题。推荐用tar xvJf命令亲测有效

安装
解压后的文件就是可执行的文件，可以开始使用了。运行start-tor-browser.desktop即可。

2.apt

教程在这里
为了文章的完整性，搬运到这里
在终端中安装

    sudo add-apt-repository ppa:webupd8team/tor-browser
    sudo apt-get update
    sudo apt-get install tor-browser

安装完成后直接运行tor-browser-en.sh即可
卸载

    sudo apt-get remove tor-browser
    rm -r ~/.tor-browser-en

链接tor网络

第一次使用tor的时候往往需要先配置
一般直接链接是连不上的，这里勾选下面的Configure

勾选yes

选择meek-amazon

勾选no
