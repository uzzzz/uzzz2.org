隐匿查询
      数据库里存储的大量的信息对标准的搜索引擎来说是不可见的，标准的搜索引擎只是索引网站上的内容，从一个链接到另一个链接。 隐匿搜索引擎专门用来搜索被称作Deep Web上的隐藏数据。

* 能查找动态数据库。
* 能在一定数据范围内查询。
* 有很好的帮助文档。
