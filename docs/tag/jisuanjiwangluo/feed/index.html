概观
Tor网络是一组志愿者操作的服务器，允许人们改善他们在互联网上的隐私和安全性。Tor的用户通过连接一系列虚拟隧道而不是直接连接来使用该网络，从而允许组织和个人通过公共网络共享信息而不损害他们的隐私。同样，Tor是一种有效的审查规避工具，允许其用户访问其他被阻止的目的地或内容。Tor还可以用作软件开发人员的构建块，以创建具有内置隐私功能的新通信工具。…
