比特币诞生于2009年，曾一度被当做极客手中的玩具，并不被重视，于是发生了程序员用1万个比特币换两个披萨的故事。如今10年过去了，比特币的身份标签已从暗网、犯罪、泡沫，逐渐变成了财富的代名词。价格则从最初的几分钱，暴涨到13万，但是这个过程却并不“顺利”。10年间，比特币经历200多次被死亡，摆脱竞争币的“围追堵截”，最终稳坐加密货币的“头把交椅”，难以被撼动。

 

比特币已经存在10年，大家都在讨论谁将是下一个区块链的风口，笔者留意到了一个叫UBTC的币，随即研究了一下这个币，发现UBTC英文全称是UnitedBitcoin，中文名是比特联储，拥有估值几千万的顶级域名UB.com。比特币元老、比特币核心开发人员、区块链领域的泰山北斗Jeff Garzik担任首席科学家及基金会董事，数字商务商会主席–Matthew Roszak担任首席策略官，团队组成堪称豪华。

 

 

 

UBTC建立的初衷是为了解决比特币日渐拥堵的问题，比特币区块链上最高时有上万笔交易积压，扩容问题一直是所有人的关注重点。UBTC在市场领先的加密货币基础上博采众长，独辟蹊径，将区块扩容至8M，加入闪电网络，在解决转账速度问题的基础之上，赋予UBTC网络智能合约，采用POW+POS混合共识机制，提出稳定货币，电子支付平台等伟大战略目标，UBTC或将成为一个真正的全球联合信用储蓄联盟。

 

【POW +POS混合共识机制】混合共识机制也被称为活动量证明，它是两种共识方法的合并。UBTC正在实施一种混合共识机制，这种机制分为工作量证明和权益证明（各50%）。一旦区块被挖出来，该过程就会重新开始。在UBTC实现这一点的方式中，没有固定的顺序，其中任何一个群体都可以挖矿。平均50％的区块将由POS挖出，50％将由POW挖出，但一个区块的后续区块可以由任意一种方法被挖出来。

【UBTC技术–智能合约】UBTC的智能合约可以说是以太坊的升级版，带来无限的可能性和创造性。UBTC使用UVM作为运行智能合约的虚拟机，而不是EVM。目前，UBTC智能合约已经正式上线，与以太坊的EVM相比，UVM旨在提供多种替代功能，优势最明显的是支持更常用的编程语言。现在想创建智能合约的人不需要学习Solidity。相反，他们可以使用自己喜欢的编程语言，比如C#、Java、Kotlin或LUA。这极大的开拓了开发者们的想象力，使智能合约编程更加简单易用。

 

【UBTC生态—支付工具】在UBTC生态中，UBPay和UUSD称得上是杀手级的应用。 UBTC的愿景是发行多种数字货币，并应用于多场景的快捷支付，UBPay就是这样一个全球性的数字支付解决方案平台，它将发行多种与传统货币锚定的数字货币，提供一种替代传统法币的支付方式。毫不夸张的认为UBPay就是打造区块链版的数字支付。使用UBPay支付平台的每一笔交易都记录在区块链上，以确保消费者和商家交易透明，有效解决欺诈、退款和支付安全的问题。

 

【UBTC生态—稳定币】 UBPay平台交易者需要一个可信赖的稳定币，来对冲加密货币市场极端的价格波动。由此，UBTC生态下的第一个稳定币UUSD应运而生。UUSD将以1:1的比例与美元挂钩（1UUSD=1美元），以满足那些要求高安全性和高透明度的人士的需求。即便所有交易均可以在区块链上查到，最重要的是，不论何时， 稳定币UUSD都会保有300%的UBTC作为抵押。

 

UBPay、UUSD将会帮助UBTC打造完整的生态闭环，一旦UBPay、UUSD开始被大规模应用，UBTC将无可限量。

UBTC于2017年12月12日面世，不到一年的时间里，UBTC技术不断突破，生态逐渐拓展，社区运营也搞得有声有色，UBTC也获得了行业的认可，目前已成功上线包括ZB（www.zb.cn）在内的18家交易所，全世界的投资者都可以通过这18家交易所自由买卖UBTC。

 

UBTC在立项之初的白皮书给出了明确的答案，目前，区块链还没有真正杀手级应用落地，还需要技术极客不断地探索。只是如果把这个过程比喻成一场马拉松，UBTC显然为未来做出了更长远的规划。UBTC的大旗必然展旗高扬！
