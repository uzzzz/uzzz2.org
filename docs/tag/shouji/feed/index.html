]]> 洋葱致西瓜——三个月纪念 https://uzzz.org/article/4080/ Tue, 16 Nov 2004 12:31:00 +0000 https://uzzz.org/article/4080/  
      西瓜和洋葱的故事是这样开始的。洋葱本来就是西瓜身边的一只大尾巴狼，表面上和善可亲，事实上心里总是打着自己的小算盘。恋爱这种事情，男生总是主动些，所以有一天大尾巴狼找了一个借口和西瓜吃饭。这一顿饭，最终成为了一场鸿门宴。饭局里，洋葱问西瓜：“你觉得我追你怎么样？”也许是因为觉得这一问题非常没有层次，问法也不够讲究，没有人愿意回答这个问题。当时的气氛很尴尬，没人知道说什么好，饭没吃完（每次和西瓜吃正餐都是要剩下东西的），两人就匆匆离开了。故事到这里似乎就要结束了，但是没有波折怎么叫好故事呢。西瓜说要逛街。后来的资料表明接下来所发生的事情不是逛街而是徒步竞走，类似长征。两人从武汉大学走到华师，再到虎泉，然后又走回来（虽然途中西瓜很累，但是坚持不搭taxi。西瓜是很有节俭原则的人。这一点俺颇为欣赏。西瓜从来不卖贵的东西。比如买手机，价格高了，她就不要了。但是不要被假象所迷惑。在西瓜的字典里，节俭的意思是这样的，比如衣服，绝对不买太贵的衣服，比如两百元以上的，但是200元以下的呢，决不手软）。途中洋葱把自己脑子里的陈芝麻烂谷子的故事全都弄出来了，还讲了一个伟大的诺贝尔得主的故事。可惜洋葱的故事大多是哲学家和数学家的，这些都超出了西瓜小脑袋的容量。更何况洋葱小学换了5个语文老师（免责声明：老师的更换和洋葱本人无关），表达能力极其有限，所以你可以经常那个看见在洋葱发表一番长篇大论之后，西瓜满脸轻松的说道，不懂。但是后来的事实证明这次徒步行走的意义是非常巨大的，它催生了洋葱和西瓜伟大友谊的萌芽，为今后友谊的发展奠定了坚实的基石。据考证，这一天是在2004年7月28日。（时间，意义都要记住，是要考试的啊）   …
