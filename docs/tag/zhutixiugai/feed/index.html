Eclipse版本：

Eclipse Java EE IDE for Web Developers.
Version: Juno Service Release 1
Build id: 20120920-0800

step1,为Eclipse添加Color.Theme的插件

这样可以方便一键更换主题，再也不用一个一个设置BackgroundColor了，同时也方便回退到default默认主题配置。

方法一：

打开Eclipse-Help-Install New Software，Add：

http://eclipse-color-theme.github.com/update

下一步，下一步，经过漫长的等待，终于可以在线下载了，这里有一个离线版本，把下载好的东西直接覆盖到Eclipse相应文件夹下，省去在线下载了，以下是传送门：

http://download.csdn.net/detail/knockheart/6690825#comment

方法二：

打开Eclipse-help-Eclipse Marketplace，在Search标签页里输入：Eclipse Color Theme 搜索：

markerplace网页在线下载（http://eclipsecolorthemes.org/?view=plugin）

如上图所示，点击install（因为我已经安装好了，所以提示uninstall），这里也可以在线下载插件，效果和方法一一样的，速度的话自行尝试。

step2，插件设置

下载好此插件后，我们Eclipse-Window-Preferences-General-Appearance，是不是多了一个Color Theme，打开看看，第一行默认就是default主题，我们可以通过这个还原回系统默认：

中间列表展示了插件自带的一些主题，右下角还有Import a theme选项，可以导入我们第三方的，具体可以通过这个地址去下载：http://eclipsecolorthemes.org/，接下来就看看我选的Zenburn主题效果：

然后我们继续看下Graphical Layout和xml的效果：

还好，xml和Graphical Layout一切正常，预览图还是像以前一样，改变的只是编程区域的主题。

step3，Eclipse四周工具栏主题修改方法

如果有完美主义者，觉得四周还是白色的，有点不协调的话，我们可以继续修改Eclipse的主题，不过有可能会导致Android中Graphical Layout，也就是xml预览图全是很色的，看不清楚。

打开Eclipse-Window-Preference-General-Appearance，右侧会看到Theme选项，一般会和你的操作系统主体一致，也是最协调的，这里默认提供的只有跟操作系统相关的，我们只能添加第三方的。下载地址：https://github.com/eclipse-color-theme/eclipse-ui-themes 然后解压拖到eclipse目录下的dropins目录

重启Eclipse，然后在Preference->General->Apperance中，选择主题为Dark Juno：

接下来看看效果：

java文件一切正常，但是发现xml中无法再看预览图了，底部的Graphical Layout找不到了，折腾半天，还是不可以，怀疑是这个Eclipse主题的问题，目前就这么多了，有更好的主题，大家可以一起分享。

不完美的解决办法，见这个帖子：https://github.com/rogerdudler/eclipse-ui-themes/issues/11#issuecomment-11439747
